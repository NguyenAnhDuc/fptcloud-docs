Relation management

1, Tạo relation Màn relation cho phép người dùng thể hiện mối quan hệ giữa các component trong một tag.

Bạn có thể kéo thả các component vào khu vực bảng để thiết lập mối quan hệ.

Mỗi component tương ứng với một node.

Ví dụ:

2, Update node

• Add value trong node

Để add value trong node, bạn thực hiện như sau:

Tại node đó, click icon Pencil >> Hiển thị hộp thoại Update node , người dùng tick vào checkbox để add value cho node đó.

Có thể add nhiều value cùng lúc.

• Remove value ra khỏi node

Trong trường hợp bạn muốn xóa bỏ một hoặc nhiều items ra khỏi node, bạn chỉ cần click vào icon Pencil để mở hộp thoại Update node và bỏ chọn items đã tick.

• Update description Để thêm hoặc chỉnh sửa mô tả cho node, bạn thao tác như sau:

Click chuột phải vào node đó, chọn Update Description. Hiển thị textbox cho người dùng nhập description.

• Disable multiple selection

Nếu người dùng chọn Disable multiple selection, thì add item vào node dưới dạng droplist, chỉ add được 1 item trong 1 node.

Để có thể add được nhiều item trong 1 node, click chuột phải và chọn Enable multiple selection.

• Delete node

Để xóa node, click chuột phải vào node đó, chọn Delete node.
