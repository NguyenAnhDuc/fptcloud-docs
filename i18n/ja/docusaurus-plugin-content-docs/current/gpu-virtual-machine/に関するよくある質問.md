# に関するよくある質問


* * *

# 一般


GPU仮想マシンまたはGPU VMは、ハードウェアリソースのGPUクラスターから作成された、専用のGPU、CPU、メモリ、ネットワークインターフェイス、ストレージを備えた独立したシステムです。 ワークロード要件に合わせて設計された、幅広い事前構成された仮想マシンが用意されており、1台のVMあたり1〜8のGPUまで柔軟なオプションを提供します。


GPU VM は、GPU、CPU、RAM、ディスクに対して事前構成されたフレーバーを提供しており、それらを要求に合わせてカスタマイズすることはできません。適切な構成を選択するか、**FPT Cloudサポート** にお問い合わせください。


GPUインスタンスが正常に作成された後にのみ、**フローティングIP** を割り当てることができます。IP構成を完了するには、IPポート(ポート転送)を構成する必要があることに注意してください。 詳細なガイドはこちらを参照してください


以下の機能は、NVIDIA Hopper(H100、H200) を搭載したGPU仮想マシンではサポートされていません。 


  * インスタンスのサイズ変更:GPUの追加または削除、CPU、RAM、ディスクのサイズ変更

  * スナップショット 

  * GPUインスタンスからテンプレートを作成する 

  * 現在、Veeamを使用したバックアップインスタンスのみがサポートされています。


  * 現在、GPUインスタンスではNVMe物理ディスクのみがサポートされています。

  * ブロックストレージを備えたGPU仮想マシンが近日中に登場します。 

  * ファイルストレージ(高パフォーマンス層)サービスがサポートされています。


NVIDIA Hopper (H100、H200)を搭載したGPU仮想マシンは、ベトナムのハノイ2 および日本の東京でご利用いただけます。


  * 予約：需要に基づいてリソースが制限され、前払い（一部または全額）または後払いで固定価格となります。請求期間は、月（3〜9）または年（1〜5）の期間になります。 

  * 従量課金制 (PAYG) :最初は無制限のリソースを利用し、後で支払うことができます。請求額は通常、日単位または時間単位で増分されます


はい、「停止」状態のインスタンスは引き続きサーバーをお客様の利用のために予約するため、このサーバーをリリースするまで料金が発生します。サーバーの料金をこれ以上請求されないようにしたい場合は、顧客向けのFPTクラウドポータルでインスタンスを削除してください。 

GPUインスタンスに接続されているその他のサービスやリソースに対しても課金されます。
