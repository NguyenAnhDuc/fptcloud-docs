# 初期設定


* * *

初めてサービスをご利用になる場合は、次の手順を確認して完了させてください。

  * **FPT Cloudアカウントの作成。** まだアカウントをお持ちでない場合は、<https://fptcloud.com/にアクセスし>、 * **「サインアップ」** をクリックして、システムの指示に従って詳細を入力します。弊社のサポートチームがすぐにご連絡し、お客様の情報を確認してアカウントを有効にいたします。

  * **ＦＰＴポータルにログイン。**<https://console.fptcloud.comにアクセスし>、 _**FPT Cloud アカウントとパスワード** でサインインします。必ず正しい_ **テナントとリージョン** を選択してください。

  * **SSHキーの設定：SSH管理** に移動してSSHキーを生成します。このキーはサーバーへの安全なアクセスに使用されます。

  * **ヘルプ** プロセス中に問題が発生した場合は、* **サポートチーム** にお問い合わせください。
