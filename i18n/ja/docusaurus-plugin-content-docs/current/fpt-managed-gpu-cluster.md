# Managed GPU Cluster (Kubernetes)


**FPTマネージドGPUクラスター** lは、FKEのマネージドGPUクラスターサービスを提供するモデルであります。MANAGED GPU CLUSTERを使用すると、FPT Cloudはすべてのコントロールプレーンのコンポーネントを管理し、ユーザーはワーカーノードをデプロイおよび管理します。MANAGED GPU CLUSTERは、ユーザーがK8sクラスターの管理にリソースを費やすことなく、アプリケーションのデプロイに集中するのに役立ちます。

**FPTマネージドGPUクラスター** は、オープンソースプラットフォームであるKubernetesに基づくサービスモデルであり、コンテナ化されたアプリケーションの展開、レプリケーション、および管理の自動化をサポートします。FPTマネージドGPUクラスターは、コンテナオーケストレーション、ストレージ、ネットワーキング、セキュリティ、PaaSの各コンポーネントを完全に統合するだけでなく、GPUリソースも提供し、複雑なコンピューティング操作をサポートします。 

**Managed GPU Cluster** を使用する前に留意すべきこと. 

  * **マネージド GPU クラスターの場所:** 地理的な場所 (リージョン) は、使用中のサーバーへのアクセス速度に影響を与える可能性があります。速度を最適化するには、トラフィックを生成するオブジェクトに最も近いリージョンを選択する必要があります。

  * **ノードの数と使用する各ノードの構成:** FPTクラウドアカウントには、RAM、CPU、ストレージ、IPリソースなどに対して一定の制限が付与されています。したがって、お客様は、FPT Cloudがお客様を最適にサポートできるように、使用するリソースと最大制限を決定する必要があります。
