# 概要


* * *

**AIノートブック とは？**

AIノートブックは、FPTのAI Factoryインフラストラクチャ上で動作するJupyterベースの開発環境です。ワンクリックで、GPU/CPUリソースと永続ストレージを備えた事前設定済みのノートブックサーバーを起動できます。

**仕組み** 各AIノートブックは、FPTのGPU/CPUインフラストラクチャに接続された、分離されたコンテナ化された環境で実行されます。ブラウザ内でコードを記述、実行、可視化するためのメインインターフェースとしてJupyterLabが使用されます。
