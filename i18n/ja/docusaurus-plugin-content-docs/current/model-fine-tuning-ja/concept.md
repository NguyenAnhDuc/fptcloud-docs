# コンセプト


* * *

FPT Model Fine-tuningは、FPT Smart Cloudによって開発されたAIモデル最適化サービスであり、ユーザーが環境のセットアップ、インフラストラクチャの管理、または複雑なコードを必要とせずに、微調整プロセスを簡素化するように設計されています。

このサービスでは、学習データセットを提供し、適切なハイパーパラメータを選択することで、特定のニーズに応じてAIモデルを簡単にカスタマイズすることができます。その後、微調整プロセスは強力なコンピューティングプラットフォーム上で自動的に実行されるため、運用作業が最小限に抑えられ、コストが接続できます。
