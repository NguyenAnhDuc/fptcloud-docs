# 監査ログ


* * *

監査ログは、微調整プロセス全体で発生する実行とパイプラインのアクティビティの包括的な記録です。監査ログを表示するには、特定のパイプラインをクリックし、[監査ログ] タブを選択してください 

このタブでは、ユーザーで検索したり、タイムスタンプ、アクション、タイプによるフィルタリングなど、検索を絞り込むためのフィルターを適用したりできます。 ![file](https://fptcloud.com/wp-content/uploads/2025/05/image-1747671875674.24.22.png)
