# 残高をチャージする


* * *

  * 下のメニューで、「課金」をクリックして課金情報と設定にアクセスします。 


![file](https://fptcloud.com/wp-content/uploads/2025/03/image-1742892495834.png)

  * モデルの微調整サービスの料金が確認できます。 ![file](https://fptcloud.com/wp-content/uploads/2025/03/image-1742892509465.png)
  * **1,500,000 VND** や**2,500,000 VND** などの推奨額をチャージするか、希望の金額を入力するかを選択できます。最低取引額は**26,000 VND** です。できたら、「購入」をクリックします。 
  * **VNPay** 支払いにリダイレクトされ、選択した金額で取引完了することができます。支払い方法は**VNPayQR** または国際支払いカードなど 


![file](https://fptcloud.com/wp-content/uploads/2025/03/image-1742892528929.png)

  * チャージ後、クレジット履歴で、チャージ取引履歴を確認します。 ![file](https://fptcloud.com/wp-content/uploads/2025/03/image-1742892550518.png)
