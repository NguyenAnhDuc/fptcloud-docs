# 接続の作成


* * *

  * ステップ1: でオブジェクトストレージ管理にアクセスし <https://console.fptcloud.com/> 、「バケット」を選択し、「バケットを作成」をクリックします。 


S3バケットの詳細については、こちらをご覧ください <https://fptcloud.com/en/documents/fpt-object-storage/?doc=tutorials-bucket>

  * **ステップ 2** ：新しいオブジェクトをバケットにアップロードします。


オブジェクトの詳細については、こちらをご覧ください <https://fptcloud.com/en/documents/fpt-object-storage/?doc=tutorials-object>

  * **_注_** ：FPT オブジェクト ストレージにオブジェクトを保存するためのバケットが正常に作成されたことを確認します。バケット作成ガイドは、こちらで参照 <https://fptcloud.com/en/documents/fpt-object-storage/>

  * **ステップ3** ：Data Hubサービスにアクセスし、接続リストに移動して、「接続を作成」をクリックします ![file](https://fptcloud.com/wp-content/uploads/2025/03/image-1742974995525.33.34.png)

  * 接続を作成するには、必要な情報を入力する必要があります。 ![file](https://fptcloud.com/wp-content/uploads/2025/03/image-1742975565417.51.05.png)

    * **名前：** 接続名を入力します 
      * **有効な文字:** 文字 (a から z、大文字と小文字を区別しない)、数字 (0から 9)、ハイフン (-) のみ。
      * **長さ:** 3〜64文字
      * **データソース：** データへの接続を有効にするためのデータソースを選択します
    * **エンドポイントURL：** 接続するバケットのエンドポイントアドレス例: <https://test-bucket-name.s3-han02.fptcloud.com>
    * **アクセスキー と 秘密キー:** アクセス許可の認証情報
  * **[保存]** をクリックして 、接続を作成します。

  * 接続が正常に作成されたら、トレーニング、評価、またはテストデータ用に 1つ選択してパイプラインを構築できます。パイプライン作成ガイドはこちらで参照できます：<https://fptcloud.com/en/documents/model-fine-tuning/?doc=create-pipeline>
