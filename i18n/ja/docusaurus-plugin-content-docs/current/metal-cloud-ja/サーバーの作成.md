# サーバーの作成


* * *

FPT カスタマー ポータル経由でベアメタルサーバーをデプロイできます。FPT は、選択した設定に基づいてプロビジョニング、OS のインストール、構成を処理します。デプロイが完了すると、SSH キーまたはパスワードを使用して SSH 経由でサーバーにアクセスできるようになります。


  9. 認証には、SSH キーまたはコンソールパスワードを選択できます。a. 利用可能な SSH キーを選択するか、新しいキーを追加します (SSH キーの作成と追加の詳細については、こちらをご覧ください)。


  11. ユーザーデータスクリプトを選択または作成します(オプション)。ユーザーデータスクリプトは、cloud-init プロセスを通じてサーバーの最初の起動時に自動的に実行されます。
