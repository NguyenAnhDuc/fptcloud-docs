# 予算アラート


**1\. 予算を作成し、アラートを設定する：**

  * ユーザーは現在まで使用した実際の予算額、または期末までの予測予算額に基づいて、定期／日次／月次／四半期／年次の予算を設定できます。

  * ユーザーは予算アラート機能の対象となるリソースグループを指定・管理できます。

  * ユーザーは目的に応じてアラートのしきい値と送信方法（システム通知または電子メールによる通知）を設定できます。


**2\. 予算履歴＆アラートを確認：**

  * ユーザーは予算グループ別に現時点までの実績コスト、実績コストと予算の差異、期末までの見積コスト、実績コストと予算の比率、見積もりコストと予算の比率等を確認できます。

  * ユーザーはアラート送信履歴およびアラート内容を追跡できます。
