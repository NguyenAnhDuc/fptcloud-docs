# サインイン


* * *

<https://marketplace.fptcloud.com> にアクセスし、「ログイン」をクリックして、FPT クラウドアカウントを入力します。

![file](https://fptcloud.com/wp-content/uploads/2025/02/image-1739520025120.png)

FPT クラウドアカウントをお持ちでない場合は、**「サインアップ」** ボタンをクリックし、指示に従ってアカウントを作成してください。

![file](https://fptcloud.com/wp-content/uploads/2025/02/image-1739520039300.png)
