# 製品情報


* * *

## 1.1 製品の詳細

ホームページまたは製品 URL から製品詳細画面にアクセスしてください。このページでは、AI モデルの詳細を確認し、推論 API のモデルとサンプルコードを試すことができます。

![file](https://fptcloud.com/wp-content/uploads/2025/02/image-1739519228034.png)

## 1.2 試してみる

チャットボックスで各モデルのテキスト生成機能をテストできます。チャットを開始するには、すでにサインインしていることでなければなりません。

## 1.3 推論 API

このタブでは、各 AI モデルと API 統合ドキュメントのインプットパラメーター、サンプルインプット/アウトプットを表示できます。

![file](https://fptcloud.com/wp-content/uploads/2025/02/image-1739760878940.png)

## 1.4 ベンチマーク

このタブでは、モデルのベンチマーク結果を見て、他のモデルと比較できます。

![file](https://fptcloud.com/wp-content/uploads/2025/03/image-1741243988861.png)

## 1.5 ライセンス

このタブでは、モデルの使用許諾契約を確認できます。

![Alt text](https://fptcloud.com/wp-content/uploads/2025/04/license.png)
