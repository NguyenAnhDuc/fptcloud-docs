# クラスターのアクセス情報を取得


* * *

Kube-configファイルは、Kubernetesクラスターへの接続情報を保存するために使用され、kubectl、kubelet、kubeadmなどのツールがKubernetes APIサーバーとの通信方法を決定するのに役立ちます。kubeconfigファイルはKubernetesへのアクセスを管理する上で重要であるため、慎重に保護する必要があります。

Kube-config ファイルをダウンロードするには、次の手順に従います。

**ステップ1: [AIインフラストラクチャ]>[マネージドGPUクラスター]** を選択するメニューで、システムに**[GPUクラスター管理]** ページが表示されます。アクセス情報を取得するクラスター を選択します。

![](https://fptcloud.com/wp-content/uploads/2025/03/12.png)

ステップ2: 基本プロパティ > クラスター情報 > 構成、kubeconfig ファイルをダウンロードして、kubectl またはその他のツールを使用してクラスターにログインできます。

![](https://fptcloud.com/wp-content/uploads/2025/03/13.png)

**注意：** マネージド GPU クラスターはネイティブ Kubernetes クラスター コアを使用し、ユーザーは通常の Kubernetes クラスターと同様に kubectl および dashboard ツールでクラスターを使用します。
