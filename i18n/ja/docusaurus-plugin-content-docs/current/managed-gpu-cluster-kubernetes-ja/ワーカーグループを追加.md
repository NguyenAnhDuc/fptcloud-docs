# ワーカーグループを追加


* * *

マネージド GPU クラスターで、ユーザーは必要に応じてクラスターにワーカー グループを追加できます。ワーカーを追加するには、ポータルで次のように操作します。

**ステップ1:** メニューで**[AIインフラストラクチャ]>[マネージドGPUクラスター]** を選択すると、**「マネージド GPU 管理」** ページが表示されます。ワーカーグループを追加するクラスターを選択します。

![](https://fptcloud.com/wp-content/uploads/2025/03/17.png)

**ステップ 2: [ノード プール] > [ワーカーの編集]** を選択します。

![](https://fptcloud.com/wp-content/uploads/2025/03/18.png)

**ステップ3: 「ワーカーグループの追加」** を選択します。

![](https://fptcloud.com/wp-content/uploads/2025/03/19.png)

**ステップ4:** 必須フィールドに入力し、**[保存]** をクリックします。

![](https://fptcloud.com/wp-content/uploads/2025/03/20.png)

  * **グループ名:** ワーカーグループを区別するために、ワーカーグループに名前を付けます
  * **コンテナランタイム:** を選択します。現在、新しいシステムはContainerdコンテナランタイムのみをサポートしています
  * **フレーバー:** ワーカーGPUのフレーバーリソース
  * **サーバー数:** クラスタのワーカーを実行するために作成されたMetalCloud サーバーの数
  * **ラベル:** ワーカー グループのラベル
  * **Taint:** Hit Taint for Worker Group (労働者グループのためのヒット テイント)


**注:** 新しいクラスタを追加するプロセスは数分かかり、クラスタのステータスが **処理中** に変わります。新しいワーカーグループを追加しても、クラスタは正常に動作します。
