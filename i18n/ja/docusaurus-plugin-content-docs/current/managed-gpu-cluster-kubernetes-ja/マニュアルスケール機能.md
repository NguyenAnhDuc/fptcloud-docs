# マニュアルスケール機能


* * *

Manual Scaleを使用すると、ユーザーはニーズに応じてシステムリソースをケーリングできます。ユーザーは、次の手順に従って、ポータルで 1 日あたりの メタルクラウドサーバーの数を増減できます。

**ステップ1:** メニューで**[AIインフラストラクチャ]>[マネージドGPUクラスター]** を選択すると、**「マネージド GPU 管理」** ページが表示されます。ワーカーグループを追加するクラスターを選択します。

![](https://fptcloud.com/wp-content/uploads/2025/03/17.png)

**ステップ2:** スケールしたいクラスターをクリックし、**[ノードプール]>[ワーカーの編集]** を選択します。

![](https://fptcloud.com/wp-content/uploads/2025/03/21.png)

![](https://fptcloud.com/wp-content/uploads/2025/03/22.png)

**ステップ3: サーバー数** 必要に応じて増やし、**[保存]** ボタンをクリックします。

![](https://fptcloud.com/wp-content/uploads/2025/03/23.png)

**注:**

サーバーのスケーリングプロセスは数分かかり、新しいワーカーがクラスターに正常に加わるまで、クラスターのステータスは**処理中** に変わります。クラスターは、新しいサーバーをスケーリングしても正常に動作します。
