# タグ 付け


  * タグは、ユーザーがリソースを管理、識別、並べ替え、検索、およびフィルタリングするのに役立ちます。ユーザーは、タグを作成して、目的別、所有者別、アプリケーション開発環境別、その他の基準など、特定のニーズに応じてリソースを分類できます。

  * 各タグは 2 つの部分で構成されています。

    * キー: タグキーの表示名。例: env、project,...

    * 値: そのキーの値。例: 「 プロダクション」 または 「 FPT 」 など


※ ユーザーはこのフィールドを空白のままにすることができます。
