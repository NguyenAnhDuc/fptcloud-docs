# インタラクティブセッションの作成


* * *


モデルテストサービスにアクセスし、「インタラクティブセッション」を選択し、「新しいセッションを作成」をクリックします ![file](https://fptcloud.com/wp-content/uploads/2025/05/image-1746688925988.21.46.png)

### インタラクティブセッションの作成

![file](https://fptcloud.com/wp-content/uploads/2025/05/image-1746689212265.22.57.png)

  * セッション名の入力 
    * デフォルトフォーマット: [int.session]_[timestamp]
    * **100文字以内** で編集可能
  * セッションの説明を入力(最大 **200文字**)
  * モデルソースオプション


セットアップを設定するときに、次の 2 つのモデル ソースから選択できます。

  * **カタログ**

    * ソースとして**カタログ** を選択すると、**モデルカタログ** から使用可能なすべてのモデルのリストが表示されます。指示モデルを選択していることを確認してください。

**例** : Llama-3.1-8B-Instruct

    * このオプションは、事前定義された公開モデルを参照または選択する場合に適しています。
  * **プライベートモデル**
    * **プライベート モデル** を選択すると、組織のカスタムモデルまたはプライベートにデプロイされたモデルのみが表示されます。
    * テストするモデルのバージョンを選択します。**モデルにブラウザファイルとタグが装備されていることを確認してください。**
    * このオプションは、カタログに公開されていないモデルを使用する場合に使用します。


完了すると、作成されたインタラクティブセッションのリストが表示されます。
