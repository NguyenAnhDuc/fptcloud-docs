# 定義


* * *

モデルテスト では、選択したモデルが特定のインプット (テキスト、画像、またはその両方) にどのように応答するかを評価できます。モデルを本番環境にデプロイしたり、アプリケーションに統合したりする前に、モデルの精度、トーン、関連性を検証する事ができます。
