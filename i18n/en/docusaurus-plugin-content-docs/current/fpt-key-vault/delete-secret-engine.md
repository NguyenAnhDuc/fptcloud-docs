# Delete Secret Engine


* * *

To delete an unused **Secret Engine** , follow these steps:

**Step 1** : Select the Key Vault to delete **Secrets Engine** on the Key Vault Management screen:

![](https://fptcloud.com/wp-content/uploads/2023/11/k4.png)

**Step 2** : In the **Secrets** tab, click **Delete** in the **Action** menu of the Secret Engine that you need to delete

![](https://fptcloud.com/wp-content/uploads/2023/11/k10.png)

**Step 3** : Click **Delete** to confirm deletion.

![](https://fptcloud.com/wp-content/uploads/2023/11/k11.png)
