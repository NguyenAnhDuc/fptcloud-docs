# Create Role


* * *

**Step 1** : Select the Auth Method to create **Role** on the Access Management screen:

![](https://fptcloud.com/wp-content/uploads/2023/11/me1.png)

**Step 2:** Click **Create**.

![](https://fptcloud.com/wp-content/uploads/2023/11/me2.png)

**Step 3** : In the **Create New Role** popup, enter the necessary information:

![](https://fptcloud.com/wp-content/uploads/2023/11/me3.png)

  * **Role Name** : The name of the Role

  * **Assigned Groups** : add Policy created in Policy Management


**Step 4** : Click **Create** to proceed. After successful creation, a popup contain **Role ID** and **Secret ID** will appear.

![](https://fptcloud.com/wp-content/uploads/2023/11/me4.png)

**Note:** To ensure safety and security, Role ID and Secret ID information are **only displayed once**. Users must store them in a safe place.
