# Create Topics


* * *

**Step 1** : From the **Application** menu > **Topics** > **Create**

![](https://fptcloud.com/wp-content/uploads/2023/11/topic_create_1.png)

**Step 2** : Enter the necessary information:

  * **Name (required)** : The name of the topic


![](https://fptcloud.com/wp-content/uploads/2023/11/topic_create_2.png)

_Note: Topic names follow the convention: t.[TOPIC_NAME]_

**Step 3** : Click OK to create the topic.
