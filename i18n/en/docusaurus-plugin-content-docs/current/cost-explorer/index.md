# Cost Explorer


Cost Explorer


:::info
🇻🇳 **Phiên bản tiếng Việt:** [Đọc bằng tiếng Việt](/cost-explorer)
:::


Cost Explorer is a tool that allows customers to customize filters by period, by resource group, or by different services, and view the cost corresponding to the filter within the last 6 months. The cost charged by your custom filter may differ from the actual cost in the month-end reconciliation period, because the time and resource group that are filtered are different from the configuration of the invoice reconciliation period.
