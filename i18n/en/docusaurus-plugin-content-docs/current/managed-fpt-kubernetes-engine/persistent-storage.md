# Persistent Storage


* * *

FPT Cloud's **M-FKE** product provides a block storage solution that supports storing data, ensuring fast read and write speeds. A **M-FKE** cluster comes with predefined storage classes offering various storage policies, each corresponding to different read and write speeds per second:

![](https://fptcloud.com/wp-content/uploads/2023/11/52-1.png)

Users can utilize these storage classes to create Persistent Volume Claims (PVC) and have applications use these PVCs.
