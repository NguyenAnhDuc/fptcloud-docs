# Tagging


Tagging


:::info
🇻🇳 **Phiên bản tiếng Việt:** [Đọc bằng tiếng Việt](/tagging)
:::


  * Tags assist users in managing, identifying, sorting, searching, and filtering resources. Users can create tags to categorize resources according to specific needs such as by purpose, by owner, by application development environment, or other criteria. 

  * Each tag consists of 2 parts: 

    * **Key** : The name displays the user's tag key. Examples: Env, Project, ... 

    * **Value** : The specific value associated with that key. For example: Production or FPT. Users can leave this field blank.
