# Create a New Webhook Secret


* * *

To configure webhooks, follow these steps:

**Step 1** : From the **Configuration** > **Webhook** screen, add new webhooks for Github, GitLab, Bitbucket, Bitbucket Server and Gogs:

  * Example for Gitlab:


![](https://fptcloud.com/wp-content/uploads/2023/11/Picture68.png)

**Step 2** : Fill in the information

  * **Secret (required)** : Secret key to perform webhook configuration with Gitlab/Github/Bitbucket/…


![](https://fptcloud.com/wp-content/uploads/2023/11/Picture69.png)

**Step 3** : Click **Create** to create the webhook

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture70.png)
