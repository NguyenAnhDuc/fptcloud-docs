# Create a new account on FPT Cloud Portal


* * *

**Step 1** : From **Security & Access** > **Local Account** > **Add Account** :

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture81.png)

**Step 2** : Fill in the information to create a new account:

  * **Username (Required)**


_Note: Username is unique, do not create duplicate usernames_

  * **Password (Required)**

  * **Capability** : Select at least one of the two tickboxes _apiKey_ or _login_

  * **Status** : Select enable or disable


![](https://fptcloud.com/wp-content/uploads/2023/11/Picture82.png)

**Step 3** : Click **Add** button to create a new account

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture83.png)

**Step 4** : Check the newly created account on AgroCD:

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture84.png)
