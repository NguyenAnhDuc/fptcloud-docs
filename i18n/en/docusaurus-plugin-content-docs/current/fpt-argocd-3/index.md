# FPT ArgoCD


FPT ArgoCD


:::info
🇻🇳 **Phiên bản tiếng Việt:** [Đọc bằng tiếng Việt](/fpt-argocd-3)
:::


FPT Smart Cloud officially launched the Managed ArgoCD Service on the FPT Cloud portal. With this service, customers can use ArgoCD to deploy applications on Kubernetes platform using kustomize, manifests, helm charts, etc. Moreover, users can easily manage ArgoCD Instance including: configuration, versions, users, etc. through the integrated interface by development team.
