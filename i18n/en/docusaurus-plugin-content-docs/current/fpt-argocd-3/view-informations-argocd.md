# View detailed informations and log in to ArgoCD instance


* * *

To work with an initialized **ArgoCD** instance, view the initialized instance's detailed information.

**Step 1:** Click on **Instance Name** to view detailed information of the selected instance.

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture58.png)

On the **Essential Properties** tab, you will be provided with detailed informations to access the **ArgoCD** cluster including the URL, and username/password to log in:

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture59.png)

**Step 2:** Log in to the **ArgoCD** cluster: From the **Essential Properties** screen, copy the URL and access the application:

  * **URL** : Information of created URL shown in **Essential Properties** tab.
  * **User** : Username shown in **Essential Properties** tab.
  * **Password** : Use the “copy” function (icon) to copy the password.


![](https://fptcloud.com/wp-content/uploads/2023/11/Picture60.png)

Interface after successfully logging in to **ArgoCD** cluster:

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture61.png)
