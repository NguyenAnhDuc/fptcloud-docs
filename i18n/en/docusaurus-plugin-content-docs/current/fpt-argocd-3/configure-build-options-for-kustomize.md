# Configure Build Options for Kustomize


* * *

To configure build options for the default kustomize build, perform the following configuration:

**Step 1** : From the configuration screen > Select **Kustomize** > **Update**

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture74.png)

**Step 2:** Fill out the **Build options** information

**Example: “—enable-helm”**

**_Note: Options are separated by spaces_**

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture75.png)

**Step 3:** Click **Save** to apply the configuration.
