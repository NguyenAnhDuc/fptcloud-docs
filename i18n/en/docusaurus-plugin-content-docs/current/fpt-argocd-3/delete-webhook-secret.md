# Delete a Webhook Secret


* * *

**Step 1:** Select **Action** > **Delete Webhook**

![](https://fptcloud.com/wp-content/uploads/2023/11/Screenshot_1.png)

**Step 2:** From the newly opened popup, type **"delete"** and click confirm to confirm deletion 

![](https://fptcloud.com/wp-content/uploads/2023/11/Picture73.png)
