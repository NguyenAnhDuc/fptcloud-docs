# Managing Account on ArgoCD


* * *

In addition to the administrator accounts, FPT Cloud supports creating and managing accounts in the system. After creating an account on the portal, by default, the user will only have the read-only permission (can view every ArgoCD information after logging in). If you want to add permissions to the created account, you need to configure the RBAC policy for the user. Refer to [Configure RBAC Policy](</en/documents/fpt-argocd-2/?doc=Configure-RBAC-policy>) in the document.
