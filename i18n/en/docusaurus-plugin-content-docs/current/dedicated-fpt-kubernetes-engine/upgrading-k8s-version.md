# Upgrading K8s Version


* * *

**Step 1** : In the FPT Cloud Portal menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page.

![](https://fptcloud.com/wp-content/uploads/2023/11/50.png)

**Step 2** : Choose a Cluster that you want to upgrade the K8s version from the list.

![](https://fptcloud.com/wp-content/uploads/2023/11/51.png)

**Step 3** : In the **Essential Properties** tab > **Cluster Information** > **Version** , click the **Setting** icon.

![](https://fptcloud.com/wp-content/uploads/2023/11/52.png)

**Step 4** : Choose the **Version** that you want to upgrade to and click **Upgrade**.

![](https://fptcloud.com/wp-content/uploads/2023/11/53.png)

**Note** : You can only upgrade but not downgrade versions.
