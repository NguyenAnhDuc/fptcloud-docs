# Increase Storage Size of Nodes


* * *

**Step 1** : In the FPT Cloud Portal menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page.

![](https://fptcloud.com/wp-content/uploads/2023/11/42.png)

**Step 2** : Choose a Cluster that you want to adjust node’s storage from the list.

![](https://fptcloud.com/wp-content/uploads/2023/11/43.png)

**Step 3** : In the **Node Pools** tab, you can expand the Storage Size of **Master Nodes** and **Worker Nodes** by clicking the **Setting** icon in the corresponding node type

![](https://fptcloud.com/wp-content/uploads/2023/11/44.png)

**Step 4** : Enter the amount of storage capacity to add > Click **Extend Disk** to make changes

![](https://fptcloud.com/wp-content/uploads/2023/11/45.png)
