# Add NFS Storage


* * *

**Step 1** : In the FPT Cloud Portal menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page.

![](https://fptcloud.com/wp-content/uploads/2023/11/54.png)

**Step 2** : Choose a Cluster that you want to add NFS Storage from the list.

![](https://fptcloud.com/wp-content/uploads/2023/11/55.png)

**Step 3** : In the **Advanced** tab > **NFS Storage** > **Status** , click the **Setting** icon

![](https://fptcloud.com/wp-content/uploads/2023/11/56.png)

**Step 4** : Enter the desired NFS Storage capacity (GB) and click **Enable** to add NFS storage.

![](https://fptcloud.com/wp-content/uploads/2023/11/57.png)
