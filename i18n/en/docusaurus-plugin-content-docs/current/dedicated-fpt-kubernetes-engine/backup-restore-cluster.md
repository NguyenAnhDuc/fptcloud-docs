# Backup & Restore Cluster


* * *

To backup the Kubernetes cluster, perform a backup of master nodes using the **Backup & Recovery** service.

**Step 1** : In the FPT Cloud Portal menu, select **Backup & Recovery**and the system will display the **Backup & Recovery** page.

![](https://fptcloud.com/wp-content/uploads/2023/11/63.png)

**Step 2** : Click **Create Backup Job**

![](https://fptcloud.com/wp-content/uploads/2023/11/64.png)

**Step 3** : The **Backup Job** screen appears. Select all Master Nodes VMs corresponding to the cluster to be backed up. Configure **Retention** and **Schedule** to suit your needs and click **Create**.

![](https://fptcloud.com/wp-content/uploads/2023/11/65.png)
