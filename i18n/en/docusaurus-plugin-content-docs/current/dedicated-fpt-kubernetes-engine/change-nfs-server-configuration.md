# Change NFS Server's Configuration


* * *

**Step 1** : In the FPT Cloud Portal menu, select **Kubernetes** , and the system will display the **Kubernetes Management** page.

![](https://fptcloud.com/wp-content/uploads/2023/11/38.png)

**Step 2** : Choose a Cluster that you want to adjust NFS Server's configuration from the list.

![](https://fptcloud.com/wp-content/uploads/2023/11/39.png)

**Step 3** : In the **Advanced** tab, you can change the Flavor (CPU, RAM) of **NFS Server** by clicking the **Setting** icon

![](https://fptcloud.com/wp-content/uploads/2023/11/40.png)

Step 4: Choose the desired configuration > Click **Change** to make changes

![](https://fptcloud.com/wp-content/uploads/2023/11/41.png)
