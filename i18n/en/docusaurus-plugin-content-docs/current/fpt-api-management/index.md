# FPT API Management


FPT API Management


:::info
🇻🇳 **Phiên bản tiếng Việt:** [Đọc bằng tiếng Việt](/fpt-api-management)
:::


**FPT API Management** is a professional API management service designed to focus on easy integration and scalability. Users can easily create, configure, and oversee their organization's APIs through the unified portal interface provided by FPT Cloud. This platform allows users to swiftly and securely deliver their organization's APIs to partners and customers, ensuring safety, control, and tight management.
