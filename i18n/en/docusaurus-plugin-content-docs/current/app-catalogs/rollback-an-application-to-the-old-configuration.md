# Rollback an Application to the old configuration


* * *

The system allows rolling-back applications to the old configuration. To do this, follow these steps:

**Step 1** : From the application details page, click **Rollback to #**

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture30-1.png)

**Step 2** : Enter **rollback** to confirm and click **Rollback** to execute the process.

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture31-1.png)

**Step 3** : View the application information after rolling-back:

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture32-1.png)
