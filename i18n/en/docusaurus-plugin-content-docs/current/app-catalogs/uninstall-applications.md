# Uninstall Applications


* * *

The system allow uninstalling installed chart. To do so, follow these steps:

**Step 1** : From the cluster detail screen, click **Uninstall**

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture35-1.png)

**Step 2** : Enter “**uninstall** ” to confirm and click **Uninstall** to execute the process.

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture36-1.png)

**Step 3** : If succeed, the system will display a notification indicates that the App Catalog has been uninstalled successfully.

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture37-1.png)
