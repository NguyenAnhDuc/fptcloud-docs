# Manage App Catalogs


* * *

**App Catalogs** allows you to manage app catalogs by repository. 

FPT Cloud supports deploying app catalogs from the Bitnami, Grafana, and FPT Cloud-managed repositories. Users can search for charts and deploy them to clusters that have been added to the app catalog
