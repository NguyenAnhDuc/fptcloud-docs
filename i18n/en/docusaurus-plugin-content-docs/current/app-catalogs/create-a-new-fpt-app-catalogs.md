# Create a new FPT App Catalogs


* * *

To create a new **App Catalog** , you can do the following:

**Step 1** : From the **FPT Cloud Portal** menu, select **App Catalog**

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture1-1.png)

**Step 2** : Click **Create App Catalog** > Enter information to create an **App Catalog Instance**

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture2-1.png)

**Step 3** : Click **Create** , and the system will initialize provisioning an App Catalog Instance and notify the processing status.

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture3-1.png)

After the **App Catalog** has been created successfully, the processing status will switch to **Succeed** , From here on, users can access the **App Catalog** Instance and manage Apps via the portal interface.

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture4-1.png)
