# FPT App Catalogs


FPT App Catalogs


:::info
🇻🇳 **Phiên bản tiếng Việt:** [Đọc bằng tiếng Việt](/app-catalogs)
:::


FPT Smart Cloud has officially launched the **FPT App Catalog** service on the portal interface. With this service, customers can ultilize the App Catalog to deploy applications through the Kubernetes platform, etc. Additionally, users can easily perform App Catalog management tasks, easily search for charts to deploy, manage applications helm charts, manage clusters, etc. through the interface that the product development team has integrated.
