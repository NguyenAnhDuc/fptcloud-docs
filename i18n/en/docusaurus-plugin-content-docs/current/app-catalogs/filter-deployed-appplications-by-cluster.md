# Filter Deployed Appplications by Cluster


* * *

**Step 1** : From the **Deployed Apps** screen, select a cluster to view the list of apps that have been deployed:

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture19-1.png)

**Step 2** : By default, the list of apps is filtered by the latest cluster that was added to the App Catalogs Management.

Cluster with deployed apps:

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture20-1.png)

Cluster without deployed apps:

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture21-1.png)

Or, clear the filter to remove the filter
