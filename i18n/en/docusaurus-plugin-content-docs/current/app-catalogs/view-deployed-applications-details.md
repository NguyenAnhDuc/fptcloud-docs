# View a Deployed Application's Details


* * *

To view detailed information about a deployed application, follow these steps:

**Step 1** : Select the deployed application in the Deployed Apps tab:

![](https://fptcloud.com/wp-content/uploads/2023/12/Picture23-1.png)

**Step 2** : Here you can view the detailed information of the app, including:

  * **App Status**

  * **Revisions List** : information about the number of deploy/upgrade

  * **Resource**

  * **Manifest**

  * **Values**


![](https://fptcloud.com/wp-content/uploads/2023/12/Picture24-1.png)
