# Reports


* * *

**Step 1***: Go to Menu > Queries & Reports

![file](https://fptcloud.com/wp-content/uploads/2025/12/image-1765765637950.png)

**Step 2** : Select Shared Groups > Report

![file](https://fptcloud.com/wp-content/uploads/2025/12/image-1765765653374.png)

By default, FPT Cloud provides built-in templates for monitoring malware detection activities, including:

  * Malware behavior reports (24h / 7 days / 30 days)

  * Exploit behavior reports (24h / 7 days / 30 days)

  * Crack-tools execution behavior reports (24h / 7 days / 30 days)


Users can customize templates by clicking New Query.
