# Update information of the WAF server


* * *

To make adjustments to a WAF server, follow the following steps:

**Step 1:** Select the WAF server that needs modification.

**![Userguide FPT WAF 2022 11](https://fptcloud.com/wp-content/uploads/2022/11/Userguide-FPT-WAF-2022-11-1024x538.png)**

**Step 2:** Edit the information of the WAF server.

User can modify information such as **Name, Description, Size**

**![Userguide FPT WAF 2022 12](https://fptcloud.com/wp-content/uploads/2022/11/Userguide-FPT-WAF-2022-12-1024x538.png)**

**Note:**

  * When changing the size of the WAF server, you are only allowed to increase the size, not decrease it.
  * Changing the WAF size may cause downtime for a few minutes until the changes are completed. You should perform these changes during low-traffic periods.

**Step 3:** Select **Update WAF** to apply the changes.
