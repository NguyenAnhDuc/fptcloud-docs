# FPT Cloud WAF


FPT Cloud WAF


:::info
🇻🇳 **Phiên bản tiếng Việt:** [Đọc bằng tiếng Việt](/fpt-cloud-waf)
:::


FPT Cloud Web Application Firewall (FPT Cloud WAF) is a firewall service designed to protect web applications against common types of attacks such as DOS (Denial of Service), Local and Remote File Inclusion, SQL Injection, Cross Site Scripting, and attacks targeting vulnerabilities in NodeJS, Java, PHP, and more.
