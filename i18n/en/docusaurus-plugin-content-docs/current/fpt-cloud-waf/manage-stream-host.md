# Manage Stream Host


* * *

Streams is a new feature of Nginx used to forward TCP/UDP traffic directly to another computer in the network.

To configure/add a new Streams, follow these steps:

**Step 1:** Select **Hosts** > **Streams** in the menu to open the Stream Host management screen.

![Userguide FPT WAF 2022 29](https://fptcloud.com/wp-content/uploads/2022/11/Userguide-FPT-WAF-2022-29-1024x503.png)

**Step 2:** Click **Create Stream** to open the screen for adding a new Stream Host and enter the information as follows:

  * **Incoming Port**
  * **Forward Host**
  * **Forward Port**


Some additional features:

  * **TCP Forwarding**
  * **UDP Forwarding**


![Userguide FPT WAF 2022 30](https://fptcloud.com/wp-content/uploads/2022/11/Userguide-FPT-WAF-2022-30-1024x502.png)

**Step 3:** Click **Create Stream** to add the new Stream Host with the entered information
