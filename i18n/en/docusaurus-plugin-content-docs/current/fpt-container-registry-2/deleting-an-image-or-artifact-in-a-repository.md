# Deleting an Image or Artifact in a Repository


* * *

If you want to delete an Image or Artifact that you pushed before to free up spaces or clean up unused projects, follow these steps:

**Step 1** : Navigate to the **Container Registry** page > from the drop-down menu in **Actions** > click **Delete**.

![](https://fptcloud.com/wp-content/uploads/2023/11/image09-1.png)

**Step 2** : Click **Delete** in the confirmation popup to confirm deletion.

![](https://fptcloud.com/wp-content/uploads/2023/11/image10-1.png)
