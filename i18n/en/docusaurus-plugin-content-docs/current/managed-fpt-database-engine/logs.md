# Event Logs


* * *

The **Event Logs** feature in FPT Database Engine allows users to monitor and control all database operations through two main types of logs:

  * **Process Log (Process Monitoring)** – Displays the detailed processing flow of system tasks.
  * **Event Logs** – Records critical events related to the lifecycle and configuration of the database, without showing internal step-by-step processing.


Key capabilities:

  * Verify whether operations have been successfully executed.
  * Track change history (audit trail).
  * Review administrative operations.


Log in to FPT Cloud Portal. From the main menu, select Database Platform → All Databases. Click the ID of the target database cluster. Switch to the “Event Logs” tab to open log management.

  * Select the “**Process Monitoring** ” tab to view process logs. Choose an action and click “**View** ” to see results.


![](https://fptcloud.com/wp-content/uploads/2024/01/view-process-monitor-scaled.png)

  * Select the “**Event Logs** ” tab to view event logs.
