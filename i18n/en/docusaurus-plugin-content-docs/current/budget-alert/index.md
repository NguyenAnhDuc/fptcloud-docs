# Budget Alert


Budget Alert


:::info
🇻🇳 **Phiên bản tiếng Việt:** [Đọc bằng tiếng Việt](/budget-alert)
:::


The Budget Alert feature allows users to deploy a proactive budget alert system based on setting time, budget, and related services on the FCI Portal.

**1\. Create a Budget and set up alerts:**

  * Allows users to set a periodic, daily/monthly, quarterly, or yearly budget based on the actual budget used to date or the projected budget to the end of the period.

  * Allow users to select and plan resource groups to alert according to set limits.

  * Allows users to set alert thresholds and how alerts are sent (via system notifications or emails) depending on the purpose of the alert.


**2\. View budget history and alerts:**

  * For each budget group, users track the cost calculated to the present time, the difference between the actual cost and the budget, the estimated cost to the end of the period, the percentage between the actual cost and the bugdet, the percentage between the expected cost and the budget.

  * Users can also view the history of sending alerts with content based on the alert threshold that has been set.
