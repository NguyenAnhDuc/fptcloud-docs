# FPT Kafka


FPT Kafka


With **FPT Kafka Service** , customers can ultilize Kafka to send and receive messages, applying it to various components in the system such as Message Queue, Data Streaming Pipeline, Data Replication Tools, etc. Additionally, users can easily perform management operations on Topics and Consumer Groups through an interface integrated by the product development team.
