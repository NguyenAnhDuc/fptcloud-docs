# Import SSL Certificate


* * *

To import an **SSL Certificate** , follow the instructions below:

**Step 1:** On the **Load Balancer** list screen, select the **Certificate** tab.

**Step 2:** Click on **Import SSL Certificate.**

Users need to enter the following information:

  * **Name** : The name of the certificate. It is recommended to use a name that helps distinguish it from other certificates, for example, the domain name of the certificate.

  * **Certificate** : The digital certificate in standard PEM format.

  * **Certificate chain** : The certificate chain in standard PEM format.

  * **Private Key** : The private key of the certificate in standard PEM format.


**Step 3:** Click **Save** to import the **SSL Certificate** or **Cancel** to abort.
