# Update Load Balancer


* * *

### Update Load Balancer

To update a **Load Balancer** , go to the **Action** section of the Load Balancer you want to update and select **Update Load Balancer**.

Here, you can update the following information:

  * **Name** : Enter a name for the **Load Balancer**.
  * **Description** : Enter a description for the **Load Balancer**.
  * **Size** : Select a **Load Balancer** package that suits your usage needs.
  * **Public IP** : Select a public IP for the **Load Balancer** if you want to access it from the internet.
  * **Subnet** : Select a subnet to place the **Load Balancer** according to your network planning. If there is no specific requirement, the system will automatically select an available subnet from the VPC.


Click **Update Load Balancer** to apply changes, or **Cancel** to discard them.
