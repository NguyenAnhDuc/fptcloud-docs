# Delete VPN Connection


* * *

To delete a VPN Connection, select **Delete** in the Action menu of the VPN you want to remove. ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744270418841.png)

A dialog box will appear — type **delete** and click **Delete VPN connection** to confirm, or click **Cancel** to discard. ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744181528169.png)
