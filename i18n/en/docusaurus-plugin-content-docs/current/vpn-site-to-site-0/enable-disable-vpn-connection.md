# Enable/Disable VPN Connection


* * *

**1\. Disable VPN Connection**

To disable a VPN connection, select **Disable VPN connection** from the Action menu. ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744269955705.png) A dialog box will appear — click **Disable VPN connection** to proceed, or **Cancel** to abort. ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744185325093.png)

> **Warning: Disable a VPN connection may take some time. Please wait until the process is completed before performing further actions.**

Check the VPN connection status once it has been successfully disabled: ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744269968890.png)

**2\. Enable VPN Connection**

To enable a VPN connection, select **Enable VPN connection** from the Action menu.

![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744269980695.png)

A dialog box will appear — click **Enable VPN connection** to proceed, or **Cancel** to abort. ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744185386777.png)

> **Warning: Enable a VPN connection may take some time. Please wait until the process is completed before performing further actions.**

Check the VPN connection status once it has been successfully enabled: ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744269992510.png)
