# Disable VPN-Site-to-Site


* * *

Disable the active VPN (if the VPN is Active). 

**Step 1** : Select the VPN connection that needs to be disabled

![Config 15](https://fptcloud.com/wp-content/uploads/2023/05/Config-15-1024x262.png)

![Disable VPN](https://fptcloud.com/wp-content/uploads/2023/05/Disable-VPN-1024x354.png)

**Step 2** : Confirm the Disable. 

![Disable VPN confirm](https://fptcloud.com/wp-content/uploads/2023/05/Disable-VPN-confirm-1024x445.png)

**Step 3** : The VPN will switch to the **Deactivating** > **Inactive state**. 

![Config inactive](https://fptcloud.com/wp-content/uploads/2023/05/Config-inactive-1024x238.png)
