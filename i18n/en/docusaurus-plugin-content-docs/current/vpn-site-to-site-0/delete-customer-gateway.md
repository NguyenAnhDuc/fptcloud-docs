# Delete Customer gateway


* * *

> **Note: A Customer Gateway can only be deleted when no VPN Connection is assigned to it (Usage VPN Connection = none).**

On the Customer Gateway list screen, click **Delete** in the Action column of the Customer Gateway you want to remove. ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744270299322.png) A dialog box will appear — type **delete** and click **Delete Customer Gateway** to confirm, or click **Cancel** to discard. ![file](https://fptcloud.com/wp-content/uploads/2025/04/image-1744183626743.png)
