# Edit VPN Information


* * *

Edit VPN Information: change VPN information after initialization.

**Step 1** : Select Edit VPN Information.

![Config edit vpn](https://fptcloud.com/wp-content/uploads/2023/05/Config-edit-vpn-1024x354.png)

**Step 2** : Change VPN information. Note, the initialized VPN information will be loaded.

![Config update](https://fptcloud.com/wp-content/uploads/2023/05/Config-update-1024x986.png)

**Step 3** : Confirm Update. 

VPN will change to Updating > Active status.

![Config active](https://fptcloud.com/wp-content/uploads/2023/05/Config-active-1024x238.png)
