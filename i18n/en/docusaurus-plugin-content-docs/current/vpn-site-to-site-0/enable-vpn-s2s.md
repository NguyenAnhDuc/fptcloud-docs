# Enable VPN-Site-to-Site


* * *

Enable: Allow VPN to work (if VPN is Inactive).

**Step 1** : Select the VPN connection to Enable.

![Config choose more action 1](https://fptcloud.com/wp-content/uploads/2023/05/Config-choose-more-action-1-1024x238.png)

![Config enable](https://fptcloud.com/wp-content/uploads/2023/05/Config-enable-1024x354.png)

**Step 2:** Confirm Enable.

![Config confỉrm enable](https://fptcloud.com/wp-content/uploads/2023/05/Config-confỉrm-enable-1024x445.png)

**Step 3** : VPN will change to **Activating** > **Active** state.

![Config active](https://fptcloud.com/wp-content/uploads/2023/05/Config-active-1024x238.png)
