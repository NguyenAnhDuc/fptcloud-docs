# Delete VPN Site-to-Site


* * *

Remove the active VPN. 

**Step 1** : Select **Delete**. 

![Config delete](https://fptcloud.com/wp-content/uploads/2023/05/Config-delete-1024x354.png)

**Step 2** : Confirm Delete. 

Record VPN will switch to Deleting and be deleted. 

![Config delete confirm](https://fptcloud.com/wp-content/uploads/2023/05/Config-delete-confirm-1024x445.png)
