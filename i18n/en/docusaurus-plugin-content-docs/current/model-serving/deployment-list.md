# Deployment List


* * *

**Step 1:** Go to **AI Platform → Model Serving → Deployment**

![Alt text](https://fptcloud.com/wp-content/uploads/2024/12/10.png) The information is displayed in a table format, listing all Deployment clusters

**Information** | **Description**  
---|---  
Model Serving Name | The name of the Model Serving  
Model Type | The type of model being deployed  
Instance Replica | The number of instances currently used in the Deployment  
Status | The status of the Deployment (refer to the Status and Action section)  
Action | Available actions for the Deployment (refer to the Status and Action section)
