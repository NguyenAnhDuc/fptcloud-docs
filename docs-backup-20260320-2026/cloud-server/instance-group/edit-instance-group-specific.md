# Chỉnh sửa Instance Group


* * *

_Tính năng chỉ áp dụng cho người dùng sử dụng dịch vụ loại Specific_

**Bước 1**. Ở menu chọn **Compute Engine** > **Instance Group** , chọn **Edit** trên instance group

![file](/img/docs/6449e439298f-image-1718788116491.png)

**Bước 2**. Chỉnh sửa thông tin instance group. Nhấn **Update** để lưu lại thông tin thay đổi. 

  * **Name** : Thay đổi thông tin tên instance

  * **Policy** : Mặc định người dùng không thể chỉnh sửa Policy

  * **Instances** : Thay đổi instance từ danh sách.

![file](/img/docs/8896d706901f-image-1718788130221.png)

**Lưu ý:**

  * **Người dùng không thể chỉnh sửa thông tin policy**

  * **Instance group có thể thay đổi instance nhưng vẫn phải đảm bảo có ít nhất 2 instance trong group**
