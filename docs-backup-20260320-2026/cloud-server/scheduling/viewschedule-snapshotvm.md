# Xem chi tiết lịch snapshot máy ảo


* * *

**Bước 1:** Trên tab Instance snapshot schedule, chọn tên của lịch cần xem.

![file](/img/docs/90c9037fa446-image-1766044475260.png)

**Bước 2:** Xem & kiểm tra thông tin chi tiết của lịch trong mục Overview bao gồm: Name, description, start & end date, frequency, tên các máy ảo gắn kèm và thông tin từng máy ảo.
