# Xem chi tiết lịch snapshot ổ đĩa


* * *

**Bước 1:** Trên tab Storage snapshot schedule, chọn tên của lịch cần xem.

![file](/img/docs/2b6c1676f27f-image-1766045049451.png)

**Bước 2:** Xem & kiểm tra thông tin chi tiết của lịch trong mục Overview bao gồm: Name, description, start & end date, frequency, tên các ổ đĩa gắn kèm và thông tin từng ổ đĩa.
