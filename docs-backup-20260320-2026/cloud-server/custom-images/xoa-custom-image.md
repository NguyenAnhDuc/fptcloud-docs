# Xóa Custom Image


* * *

Nếu không còn nhu cầu sử dụng, người dùng có thể xóa **Custom Image** theo hướng dẫn sau: 

**Bước 1:** Ở menu chọn **Custom Image** , trong phần **Actions** của Image cần xóa, chọn **Delete**. ![file](/img/docs/61bbf341d0eb-image-1734596886940.png) **Bước 2:** Hệ thống sẽ hiển thị popup xác nhận thông tin xóa, để đồng ý xóa chọn **Delete custom image**. ![file](/img/docs/3da0ab77e4c4-image-1734596914709.png)
