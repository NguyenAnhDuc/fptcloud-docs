# 2. Hướng dẫn sử dụng Grafana


Hướng dẫn sử dụng Grafana

Tài liệu này hướng dẫn người dùng cách truy cập và sử dụng Grafana Dashboard trong hệ thống FMON – FPT Monitoring nhằm theo dõi (monitoring) trạng thái hoạt động của các dịch vụ trong hệ thống.

Nội dung tập trung vào:

  * Truy cập và sử dụng Grafana Dashboard

  * Tích hợp Monitoring cho các dịch vụ như Kubernetes (K8S), Database, …

  * Xem và phân tích metrics, logs, traces, alerts

  * Hỗ trợ người dùng nhanh chóng phát hiện, phân tích và xử lý sự cố trong quá trình vận hành.
