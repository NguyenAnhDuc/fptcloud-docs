# 2. FPT Cloud Guard


Chỉnh sửa Alert


* * *

**Bước 1** : Ở menu chọn **Cloud Guard** , chọn tab **Resource alert**

![file](/img/docs/6dfc0d32a612-image-1725532789529.png)

**Bước 2** : Chọn Alert cần chỉnh sửa. Hệ thống sẽ hiển thị màn hình Update.

![file](/img/docs/3d095f595b4d-image-1725532811064.png)

**Bước 3** : Nhập thông tin mới vào các trường tương ứng và chọn **Update**.

Lưu ý: FPT Cloud Guard chỉ hỗ trợ người dùng chỉnh sửa một số trường thông tin cần thiết
