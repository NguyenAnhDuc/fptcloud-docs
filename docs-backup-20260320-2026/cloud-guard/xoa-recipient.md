# 2. FPT Cloud Guard


Xóa Recipient


* * *

**Bước 1** : Trên bảng quản trị **Recipient** **Management** , trong phần **Action** của Recipient cần xóa, chọn **Delete**.

![file](/img/docs/864680e6b1ec-image-1725533165171.png)

**Bước 2** : Một hộp thoại cảnh báo sẽ hiện lên, hiển thị tên Recipient và yêu cầu người dùng xác nhận. Chọn **Delete** để tiến hành xóa.
