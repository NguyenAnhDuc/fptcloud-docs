# 2. FPT Data Platform


Jupyterhub


* * *

**Jupyterhub** là một nền tảng mã nguồn mở được thiết kế để cung cấp môi trường Jupyter Notebook đa người dùng, cho phép các nhà khoa học dữ liệu, kỹ sư dữ liệu, và nhà phát triển phần mềm truy cập vào tài nguyên tính toán để phân tích, xử lý dữ liệu và phát triển mô hình học máy. Khi tích hợp vào Cloud Data Platform, JupyterHub trở thành một thành phần cốt lõi, cho phép quản lý, mở rộng và tối ưu hóa tài nguyên trên các dịch vụ cloud, từ đó hỗ trợ quá trình lưu trữ, xử lý dữ liệu lớn.
