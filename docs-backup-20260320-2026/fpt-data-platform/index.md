# 2. FPT Data Platform


FPT Data Platform

**FPT Data Platform** là sản phẩm tạo dựng các bộ công cụ để tích hợp dữ liệu. Mục tiêu của **FPT Data Platform** là tạo ra một cơ sở dữ liệu hoặc hệ thống dữ liệu thống nhất và đồng nhất, giúp tổ chức hiểu được dữ liệu của mình một cách toàn diện và sử dụng nó để hỗ trợ quyết định kinh doanh, phân tích dữ liệu và các mục tiêu khác.
