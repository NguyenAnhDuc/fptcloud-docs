# 2. Relation management


Dashboard


* * *

Để tạo một dashboard, người dùng thực hiện như sau:

**Bước 1** : Vào Home >> Dashboard >> New dashboard

**Bước 2** : Người dùng sẽ add variable trong dashboard bằng cách click icon Settings >> Chọn Variables >> Add variables

**Bước 3** : Khai báo tag có trong dashboard. 

![Alt text](/img/docs/c72550b8454a-Picture30.png)

Sau khi tạo xong sẽ có một variable được khai báo và hiển thị ở màn hình.

![Alt text](/img/docs/b13fb68996f2-Picture31.png)

**Bước 4** : Tạo các biến tương ứng với các component bên trong tag.

![Alt text](/img/docs/152d530014af-Picture33.png)

Click Apply để lưu biến.

Các biến đã khai báo sẽ được hiển thị ở màn hình.

![Alt text](/img/docs/86ebccbf4943-Picture35.png)

**Bước 5** : Save dashboard. Quay lại trang dashboard bằng cách chọn Home >> Dashboard >> Chọn dashboard đã tạo

Trên dashboard lúc này sẽ hiển thị các variables mà người dùng đã khai báo.

![Alt text](/img/docs/abda8417a8ec-Picture36.png)

Bạn có thể add panel theo mong muốn để theo dõi.
