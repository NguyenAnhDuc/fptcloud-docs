# 2. Network Interface Card (NIC)


Xóa Network Interface Card


* * *

**Bước 1** : Chọn Card NIC cần xóa. Đối với card Primary, chức năng này sẽ bị disable. Đối với card thông thường, Unify Portal hỗ trợ xóa khi không có nhu cầu sử dụng.

![file](/img/docs/f9a3bcaf657d-image-1691389181936.png)

![file](/img/docs/69a54bea54e8-image-1691389200234.png)

**Bước 2** : Xác nhận card cần xóa.

![file](/img/docs/c6e902c674b7-image-1691389217097.png)

**Bước 3:** Chờ hệ thống xử lý.

![file](/img/docs/40693d1594d3-image-1691389227628.png)

**Bước 4:** Card được xóa thành công và Instance sẽ không thể kết nối qua card này. Để đảm bảo thông tin đồng bộ, các bạn cần thay đổi thông tin về network trong file config của hệ điều hành. Link hướng dẫn nhanh tại: [**How to change Config file**](<https://fptcloud.com/documents/change-network-config-0/>).
