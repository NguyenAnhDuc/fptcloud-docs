# GPU Virtual Machine

GPU Virtual Machine (GPU VM) là dịch vụ máy chủ ảo được trang bị GPU của FPT Cloud, phù hợp cho các tác vụ AI/ML, deep learning, rendering và xử lý đồ họa hiệu suất cao.

Tham khảo tài liệu tiếng Anh để biết hướng dẫn chi tiết.
