# 2. FPT Autoscale


Đổi tên Profile


* * *

## **Bước 1** : Truy cập trang **Autoscaling > Autoscale Profile**. Click vào tên profile cần đổi tên.

![click to rename profile](/img/docs/e3a959ae83ad-Screenshot-2024-10-01-164339.png)

## **Bước 2** : Nhập tên mới cho profile và chọn **Update profile** để xác nhận.

![rename profile](/img/docs/9da981c0d5fc-screencapture-console-fptcloud-000823-INFRA-AUTOMA.png)

Sau khi cập nhật thành công, profile hiển thị trong danh sách với tên mới:

![list profile after rename](/img/docs/fe6bd0c40085-Screenshot-2024-10-01-164932.png)
