# 2. FPT Backup As a Service


Create Veeam Recovery Media


* * *

**Bước 1:** Khởi động ứng dụng Create Recovery Media. ![file](/img/docs/75c1dd9651b4-image-1695292809658.png)

**Bước 2:** Cửa sổ hiện ra, tùy chọn các kết nối mạng, driver mà bạn muốn tích hợp vào file Veeam Recovery Media sau đó bấm Next để tiếp tục. ![file](/img/docs/6d9e55d8922b-image-1695292817914.png)

**Bước 3:** Đặt tên và chọn nơi lưu trữ file ISO. Sau đó bấm Next để tiếp tục.

  * Lưu ý không nên lưu file Veeam Recovery Media tại server cần backup, để tránh trường hợp khi server bị xóa hoặc không thể khởi động bạn không thể lấy file này để sử dụng. ![file](/img/docs/585e6c605bbf-image-1695292826165.png) **Bước 4:** Kiểm tra lại thông tin sau đó bấm Create. ![file](/img/docs/81e26e7e2f3b-image-1695292833515.png) **Bước 5:** Sau khi hoàn thành việc khởi tạo, bấm Finish để kết thúc. ![file](/img/docs/d35664ce9110-image-1695292841371.png) **Veeam Recovery Media for Linux:** Bước 1: Đối với Linux bạn có thể download trực tiếp tại link bên dưới. <https://download2.veeam.com/VAL/v5/veeam-recovery-media-5.0.0.4318_x86_64.iso>
