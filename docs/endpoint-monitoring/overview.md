# 2. Endpoint monitoring


Overview


* * *

**Bước 1** : Truy cập thành công vào dịch vụ 

**Bước 2** : Trên menu của **Endpoint monitoring** , click **Overview**

Hiển thị màn overview theo dõi giám sát tất cả các endpoint đã khai báo 

Hình ảnh minh họa: 

![](/img/docs/b340939483c4-Screenshot_18.png)

Các thông tin trong màn overview bao gồm: 

  * Bảng Bare Metal Server Probes (Up/Down) - Current: thông tin các endpoint UP/DOWN ở thời điểm hiện tại 

  * Bare Metal Server Probes (Up/Down) - Historic 

  * SSL Cert Expiry 

  * DNS Lookup 

  * Probe Duration
