# 2. Endpoint monitoring


Xem danh sách endpoints


* * *

**Bước 1** : Truy cập thành công vào dịch vụ 

**Bước 2** : Trên menu của dịch vụ, click **Endpoints**

**Bước 3** : Tại đây hiển thị danh sách các endpoint đã tạo kèm theo các thông tin như status endpoint, type, ngày tạo scrape interval... 

![](/img/docs/f4d57ebf3359-Screenshot_12.png)

**Bước 4** : Để xem chi tiết monitor của từng endpoint, bạn click vào tên endpoint để điều hướng tới trang dashboard để theo dõi endpoint. 

![](/img/docs/3eda204aca7d-Screenshot_13.png)

Danh sách metric có thể theo dõi, gồm có: 

  * Status: up / down 

  * HTTP status code: 2xx, 3xx, 4xx, ... 

  * HTTP version: 1.0, 1.1, 2.0 

  * SSL support 

  * SSL expiry 

  * HTTP duration 

  * DNS lookup 

  * Probe Duration
