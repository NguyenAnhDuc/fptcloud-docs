# 2. FPT Cloud Monitoring


7.1. Truy cập vào dashboard


* * *

**Bước 1** : Đăng nhập vào FPT portal cloud **<https://console.fptcloud.com>**

**Bước 2** : Trên menu của FPT Cloud Portal, click **Monitoring**. 

**Bước 3** : Click chi tiết vào một Workspace, sau đó di chuyển tới **Configuration Data** tab.

**Bước 4** : Click vào dashboard url, dashboard sẽ tự động login bằng Single Sign On (SSO)
