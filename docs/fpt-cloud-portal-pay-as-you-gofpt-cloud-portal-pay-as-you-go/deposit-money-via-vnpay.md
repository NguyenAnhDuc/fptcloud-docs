# Nạp tiền thông qua VNPAY


* * *

User có thể nạp tiền thông qua cổng thanh toán VNPAY

  * B1. Vào trang Billing, chọn menu Topup. ![file](/img/docs/dcc595d7d548-image-1694073789071.png)
  * B2. Chọn mệnh giá tiền muốn nạp hoặc nhập số tiền muốn nạp ![file](/img/docs/6af15e22f348-image-1694073800926.png)
  * B3. Bấm nút Pay via VNpay và hoàn thành quy trình thanh toán của VNPAY 
    * Nếu thanh toán thành công, hệ thống sẽ hiển thị thông báo và cộng tiền vào tài khoản ![file](/img/docs/e6b48ffa0e72-image-1694073814699.png)
