# Mua storage disk (ổ đĩa rời)


* * *

Khách hàng đã nạp tiền trên hệ thống có thể mua ổ đĩa rời.

  * B1. Vào trang Compute Engine - Storage Disks ![file](/img/docs/97d410095cb4-image-1694073928674.png)
  * B2. Chọn New Storage ![file](/img/docs/c9e0ea1f9b91-image-1694073937844.png)
  * B3. Chọn Storage policy và size 
    * Hệ thống estimate số tiền sử dụng storage trong 1 giờ cho khách hàng.
  * B4. Bấm Create storage disk

    * Nếu các thông tin hợp lệ thì hệ thống tạo storage và đi đến trang Storage disk
    * Nếu user đã sử dụng hết giới hạn tài nguyên, hệ thống sẽ báo lỗi ![file](/img/docs/7a7b3dacb585-image-1694073957418.png)

Lưu ý: 

  * Khách hàng cần có ít nhất 50.000đ + tiền sử dụng storage trong 1h để khởi tạo
  * Hệ thống sẽ trừ tiền chi phí sử dụng storage trong 1h ngay khi khởi tạo. ![file](/img/docs/6d3c6da14389-image-1694073966294.png)
