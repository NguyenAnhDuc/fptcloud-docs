# 2. FPT Event Gateway


Cập nhật thông tin Router group 


* * *

Để cập nhật thông tin Router group, người dùng thực hiện các bước sau: 

**Bước 1:** Tại thanh menu chọn **Integration** > chọn **Event Gateway**

**Bước 2:** Tại danh sách **Event Gateway** > chọn **Event Gateway** > chọn **tab Security**

**Bước** 3: Tại danh sách **Router group** > chọn **Router group** cần cập nhật thông tin chi tiết 

**Bước 4** : Chọn biểu tượng Edit tại mục cần cập nhật thông tin. 

  * **Popup: Update Basic Information**

  * **Name (required):** Tên **Router group**

    * Bao gồm a-z, A-Z, -, _ và số; bắt đầu bằng chữ cái; tối đa 30 ký tự 
  * **Description** (optional): Mô tả 

![Hình ảnh](/img/docs/aadfbc716452-image23.png)

  * **Popup: Update routers**

    * Hiển thị popup cho phép thêm hoặc xóa các router trong group. 

![Hình ảnh](/img/docs/bd82d929b6c9-image24.png)

**Bước 5** : Tại các popup, sau khi cập nhật, chọn **Save** để lưu các thay đổi
