# 2. Dedicated – FPT Kubernetes Engine


Tăng Storage size của NFS Server


* * *

**Bước 1** : Ở menu chọn **Kubernetes** , hệ thống sẽ hiển thị trang **Kubernetes Management**.

![](/img/docs/c657a04c7116-Picture30.png)

**Bước 2** : Chọn **Cluster** muốn thay đổi cấu hình nodes trên danh sách.

![](/img/docs/f3d03b46eaeb-Picture31.png)

**Bước 3** : Chọn tab **Advanced** , bạn có thể thay đổi Storage size của *_NFS Server_ bằng cách chọn icon Setting ở config tương ứng.

![](/img/docs/973e4b6709e2-Picture48.png)

Bước 4: Nhập dung lượng cần tăng thêm > Chọn **Extend** để thực hiện tăng dung lượng storage.

![](/img/docs/e1270d2161ac-Picture49.png)
