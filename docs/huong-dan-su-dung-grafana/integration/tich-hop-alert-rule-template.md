# 2. Hướng dẫn sử dụng Grafana


3.2. Tích hợp Alert rule template


* * *

**Bước 1:** Tại giao diện Grafana, mở sidebar Menu 

![Alt text](/img/docs/a5fead322aff-Picture3.png)

**Bước 2:** Mở mục **Apps** và chọn **Fmon – Templates**

![Alt text](/img/docs/4e0e3943d8c0-Screenshot_7.png)

**Bước 3:** Tại giao diện Fmon – Templates, chọn tab **Alert rule template** và chọn **Topic** cho dịch vụ cần thêm Alert Rule

![Alt text](/img/docs/ca3c0b947d8f-Screenshot_8.png)

**Bước 4:** Ấn **Integrate** từng Alert rule mong muốn

![Alt text](/img/docs/cc567b1ae52a-Screenshot_9.png)

**Bước 5:** Khi trạng thái tích hợp Alert Rule chuyển sang **Integrated** , Alert Rule mong muốn đã được thêm vào mục Alert Rules của KH

![Alt text](/img/docs/3b3797a40bab-Screenshot_10.png)
