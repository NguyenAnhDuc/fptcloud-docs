# 2. Hướng dẫn sử dụng Grafana


4.4. Xem thông tin trace


* * *

**Bước 1:** Trên menu, bạn chọn **Explore**

![Alt text](/img/docs/f2b7b75d09ea-Screenshot_22.png)

**Bước 2:** Chọn datasource: **tempo**

Hệ thống mở tab **TraceQL** cho phép bạn nhập câu query

![Alt text](/img/docs/a8f6ab7df8a6-Screenshot_29.png)

**Bước 3:** Nhấn _Run query_ để hiển thị kết quả

_Ảnh minh họa_

![Alt text](/img/docs/e54fb90fced9-Screenshot_30.png)

Hướng dẫn chi tiết, vui lòng tham khảo tài liệu chính thức của Grafana [TẠI ĐÂY](<https://grafana.com/docs/tempo/latest>)
