# 2. Hướng dẫn sử dụng Grafana


4.5. Hướng dẫn Query Metric Data


* * *

Phần này dành cho người dùng có nhu cầu tự xây dựng truy vấn.

**Bước 1:** Trên menu Grafana chọn **Explore**

![Alt text](/img/docs/f2b7b75d09ea-Screenshot_22.png)

**Bước 2:** Chọn datasource, ví dụ: loki

![Alt text](/img/docs/0fcd00d8ce9b-Screenshot_31.png)

**Bước 3:** Chọn chế độ code >> Nhập query

**Bước 4:** Nhấn button **Run query** ở góc phải màn hình.

Màn hình trả ra kết quả minh họa như sau:

![Alt text](/img/docs/4151422dad57-Screenshot_35.png)
