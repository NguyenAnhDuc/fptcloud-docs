# 2. FPT Monitoring – IP Access Control


Truy cập tính năng và kiểm tra trạng thái bật/tắt


* * *

  * Trên portal FPT Cloud Console, vào workspace FPT Cloud Monitoring cần cấu hình, chọn tab **“IP Access Control”**. 

  * Nếu tính năng chưa bật, bạn sẽ thấy nút “Enable IP Access Control". Sau đó nhấn nút “Enable IP Access Control” để bật tính năng.

![Enable IP Access Control](/img/docs/37902d64f2e9-Screenshot_2.png)

  * Nếu tính năng đã bật, bạn sẽ thấy 4 bảng whitelisted IP tương ứng với 4 thành phần hệ thống FPT Cloud Monitoring, và nút “Disable IP Access Control". Nhấn nút “Disable IP Access Control” để tắt tính năng.

![Màn hình chính](/img/docs/2a6306f9b98d-Screenshot_3.png)

**Lưu ý:**

  * **Khi tính năng IP Access Control tắt, mọi IP đều có thể truy cập vào các thành phần hệ thống của workspace FPT Cloud Monitoring.**

  * **Ngay khi bật tính năng, mặc định hệ thống sẽ chặn hết mọi truy cập từ bên ngoài FPT Cloud (nghĩa là các agent monitoring trên FPT Cloud vẫn hoạt động bình thường)**. **Các IP từ bên ngoài FPT Cloud cần được whitelist để có thể truy cập hệ thống FPT Cloud Monitoring.**
