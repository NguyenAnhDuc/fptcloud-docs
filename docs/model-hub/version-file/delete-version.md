# 2. Model Hub


Delete version


* * *

Xóa phiên bản là xoá thông tin và các tệp của phiên bản trong hệ thống 

Để xóa phiên bản, bạn thao tác như sau: 

Bước 1: Click vào 1 model để xem chi tiết > Chọn tab Version 

Bước 2: Click vào dấu 3 chấm ở cột Action > Chọn Delete ![](/img/docs/162129aecb78-Picture18.png) Bước 3: Hiển thị hộp thoại xác nhận, để đồng ý xóa, bạn nhập “delete” vào text box. Lúc này button Delete sẽ enable lên cho người dùng xóa. ![](/img/docs/604dc55cb29a-Picture19.png) Bước 4: Để xóa, click button Confirm, để hủy bỏ thao tác chọn Cancel
