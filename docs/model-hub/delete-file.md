# 2. Model Hub


Delete file


* * *

Mục đích: Xoá các tệp tương ứng với version 

Các bước thực hiện: 

Bước 1: Click vào 1 model để xem chi tiết > Chọn tab File Browser 

Bước 2: Click vào dấu 3 chấm cột Action > Chọn Delete ![](/img/docs/43e776d4e3b2-Picture25.png) Bước 3: Hiển thị hộp thoại xác nhận, để đồng ý xóa, bạn nhập “delete” vào text box. Lúc này button Delete sẽ enable lên cho người dùng xóa. ![](/img/docs/b12a6e4df591-Picture26.png) Bước 4: Để xóa, click button Confirm, để hủy bỏ thao tác chọn Cancel
