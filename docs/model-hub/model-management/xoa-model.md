# 2. Model Hub


Xóa model


* * *

Mục đích: xóa 1 model khỏi danh sách các model đã tạo 

Để xóa model, bạn thực hiện theo các bước sau: Bước 1: Tại màn hình Model list, click vào title của Model mà bạn muốn xóa, màn hình đi đến màn Model detail 

Bước 2: Tại đây, bạn click button Actions, chọn action Delete ![](/img/docs/d464a2e0760d-Picture14.png) Bước 3: Hiển thị hộp thoại xác nhận, để đồng ý xóa, bạn nhập “delete” vào text box. Lúc này button Delete sẽ enable lên cho người dùng xóa ![](/img/docs/64ad087837de-Picture15.png) Bước 4: Click button Confirm để xóa, để hủy bỏ thao tác chọn Cancel
