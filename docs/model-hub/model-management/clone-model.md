# 2. Model Hub


Clone model (v1.1)


* * *

Mục đích: Nhân bản 1 model giống với model đã có.

Các bước thực hiện: 

Bước 1: Click vào 1 model để xem chi tiết 

Bước 2: Click vào Actions > Chọn action Clone

![](/img/docs/d1085098af50-anh-7.png) Bước 3: Sau khi click Clone, hiển thị màn hình clone model với các thông tin là thông tin của model gốc mà người dùng đang thực hiện clone. Tại đây, người dùng thực hiện chỉnh sửa lại Model title để đảm bảo model name là duy nhất, điền các thông tin cần thiết khác 

Bước 4: Click Save. Hệ thống trở lại màn hình model list với model mới vừa thực hiện clone
