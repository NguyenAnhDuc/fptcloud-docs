# 2. CheckPoint – FPT NGFW


Tạo Object


* * *

**Object** sẽ định nghĩa các máy chủ, dải mạng, port, … được sử dụng cho việc thiết lập rule.

**Bước 1:** Trên **Object Pane** (**F11**) > chọn **New** > chọn **Network**.

![Userguide Checkpoint FPT NGFW 2022 44](/img/docs/e9586759cc22-Userguide-Checkpoint-FPT-NGFW-2022-44-1024x333.png)

**Bước 2:** Điền thông tin name, address, mask, comment về dải mạng muốn tạo > chọn **OK.**

![Userguide Checkpoint FPT NGFW 2022 45](/img/docs/721733e9b581-Userguide-Checkpoint-FPT-NGFW-2022-45.png)

**Bước 3:** Install Policy.

**Bước 1:** Trên **Object Pane** (**F11**) > chọn **New** > chọn **Host.**

![Userguide Checkpoint FPT NGFW 2022 46](/img/docs/6a86ce6740fe-Userguide-Checkpoint-FPT-NGFW-2022-46-1024x261.png)

**Bước 2:** Điền thông tin name, IPv4 address, comment về dải mạng muốn tạo > chọn **OK.**

![Userguide Checkpoint FPT NGFW 2022 47](/img/docs/6603e0e1d533-Userguide-Checkpoint-FPT-NGFW-2022-47.png)

**Bước 3:** Install Policy.

**Bước 1:** Trên **Object Pane** (**F11**) > chọn **New** > chọn **More** > chọn **Service** > chọn loại service muốn tạo (**UDP/TCP/RPC/…**)

![Userguide Checkpoint FPT NGFW 2022 48](/img/docs/caf6afa6bdc9-Userguide-Checkpoint-FPT-NGFW-2022-48-1024x287.png)

**Bước 2:** Điền thông tin name, port, comment về service muốn tạo > chọn **OK.**

![Userguide Checkpoint FPT NGFW 2022 49](/img/docs/adcedff8f39c-Userguide-Checkpoint-FPT-NGFW-2022-49.png)

**Bước 3:** Install Policy.
