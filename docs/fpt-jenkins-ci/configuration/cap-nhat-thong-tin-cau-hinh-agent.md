# 2. FPT Jenkins CI


Cập nhật thông tin cấu hình Agent


* * *

Để cập nhật thông tin cấu hình Agent người dùng thực hiện như sau:

**Bước 1.** Tại menu FPT Portal > **Jenkins CI** > **Detail Cluster** > **Agent Managements** > Click **Actions** đối với agent muốn cập nhật: ![](/img/docs/519a8553d0f8-Picture19.png) **Bước 2.** Chọn **Edit** để thực hiện chỉnh sửa cấu hình cho Agent: ![](/img/docs/4d54db71faa6-Picture20.png) **Bước 3.** Thực hiện chỉnh sửa thông tin cấu hình: Lựa chọn tài nguyên muốn update ![](/img/docs/e8aaf98b1ccd-Picture21.png) **Bước 4.** Nhấn **Update** để thực hiện cập nhật thông tin cấu hình. ![](/img/docs/ec7e12428a6f-Picture22.png) **Lưu ý** : Chỉ được Update lên tài nguyên cao hơn
