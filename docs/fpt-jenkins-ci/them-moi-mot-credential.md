# 2. FPT Jenkins CI


Thêm mới một Credential


* * *

Với mỗi loại credential, người dùng có thể thực hiện cấu hình như sau:

**Bước 1.** Tại menu FPT Portal > **Jenkins CI** > **Detail Cluster** > **Credential Managements** : ![](/img/docs/f1ca981e56e1-Picture28.png)

  * **Tạo mới Credential loại Secret Text**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **Secret Text** : ![](/img/docs/4e9193293311-Picture29.png) **Bước 3.** Nhập thông tin để tạo credential với kind Secret Text: ![](/img/docs/6ec104f72a00-Picture30.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind Secret Text. Kết quả sau khi tạo credential: ![](/img/docs/15489933d450-Picture31.png) Kết quả tạo credential dưới Jenkins instance: ![](/img/docs/690a204c23c0-Picture32.png) ![](/img/docs/c7a729fdec5f-Picture33.png)

  * **Tạo credential với loại Username with password**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **Username with password** : ![](/img/docs/0260b9d4b35c-Picture34.png) **Bước 3**. Nhập thông tin để tạo credential với kind **Username with password** : ![](/img/docs/52ee14794240-Picture35.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind **Username with password**. Kết quả sau khi tạo credential: ![](/img/docs/ce02107a2d28-Picture36.png) Kết quả tạo credential dưới Jenkins instance: ![](/img/docs/42448ed61f59-Picture37.png) ![](/img/docs/e8817bc13354-Picture38.png)

  * **Tạo credential với loại GitLab API token**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **GitLab API token** : ![](/img/docs/5f3a117bdcdf-Picture39.png) **Bước 3.** Nhập thông tin để tạo credential với kind **GitLab API token** : ![](/img/docs/bd9447846e80-Picture40.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind **GitLab API token**. Kết quả sau khi tạo credential: ![](/img/docs/232f76192acb-Picture41.png) Kết quả tạo credential dưới Jenkins instance: ![](/img/docs/addc31f00a8c-Picture42.png) ![](/img/docs/4d3ae4c9aebd-Picture43.png)

  * **Tạo credential với loại SSH Username with private key**

**Bước 2.** Tại màn hình **Credential Managements** > **Add Credential** , thực hiện thêm mới một credential loại **SSH Username with private key** : ![](/img/docs/cc87bfd1413d-Picture44.png) **Bước 3.** Nhập thông tin để tạo credential với kind **SSH Username with private key** : ![](/img/docs/562215f21e15-Picture45.png) **Bước 4.** Nhấn **Add** để tạo một credential với kind **SSH Username with private key**. Kết quả sau khi tạo credential: ![](/img/docs/aa140b9e4673-Picture46.png) Kết quả tạo credential dưới Jenkins instance: ![](/img/docs/03acc4a6f8ae-Picture47.png) ![](/img/docs/8be15bf657fd-Picture48.png)
