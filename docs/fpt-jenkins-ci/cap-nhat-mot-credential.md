# 2. FPT Jenkins CI


Cập nhật một credential


* * *

Để cập nhật thông tin một credential, người dùng có thể thực hiện như sau:

**Bước 1.** Tại menu FPT Portal > **Jenkins CI** > Detail Cluster > **Credential Managements:**

![](/img/docs/64efc5d7bd05-Picture49.png)

  * **Cập nhật Credential loại Secret Text**

**Bước 2.** Tại màn **Credential Managements** > Chọn credential loại Secret Text cần Edit > Actions **Edit** : ![](/img/docs/0e401103b2ac-Picture50.png) **Bước 3.** Thực hiện chỉnh sửa các thông tin: ![](/img/docs/dbcf07bc5ae9-Picture51.png) **Bước 4.** Kết quả sau khi cập nhật thông tin ![](/img/docs/7c7e78e01bdc-Picture52.png)

  * **Cập nhật Credential loại Username with password**

**Bước 2.** Tại màn **Credential Managements** > Chọn credential loại Username with password cần Edit > Actions **Edit** : ![](/img/docs/c95f42681454-Picture53.png) **Bước 3.** Thực hiện chỉnh sửa các thông tin: ![](/img/docs/0edd4becef33-Picture54.png) **Bước 4.** Kết quả sau khi cập nhật thông tin ![](/img/docs/cda816032464-Picture55.png)

  * **Cập nhật Credential loại GitLab API token**

**Bước 2.** Tại màn **Credential Managements** > Chọn credential loại GitLab API token cần Edit > Actions **Edit** : ![](/img/docs/c87fb77345de-Picture56.png) **Bước 3.** Thực hiện chỉnh sửa các thông tin: ![](/img/docs/23d42fb31bf5-Picture57.png) **Bước 4.** Kết quả sau khi cập nhật thông tin ![](/img/docs/0f86bc092252-Picture58.png)

  * **Cập nhật Credential loại SSH Username with private key**

**Bước 2.** Tại màn **Credential Managements** > Chọn credential loại SSH Username with private key cần Edit > Actions **Edit** : ![](/img/docs/5564d869b4c6-Picture59.png) **Bước 3.** Thực hiện chỉnh sửa các thông tin: ![](/img/docs/b5e5b888ad9f-Picture60.png) **Bước 4.** Kết quả sau khi cập nhật thông tin ![](/img/docs/37e0a81e9c35-Picture61.png)
