# 2. FPT Jenkins CI


Upgrade version


* * *

FPT Cloud hỗ trợ người dùng upgrade phiên bản Jenkins lên phiên bản mới hơn trong số các phiên bản FPT Cloud support. Hiện tại trên Portal đang support một số phiên bản bao gồm: 2.452,2.453,2.454.

**Bước 1.** Vào menu **Essential Properties** > **Instance Information** ![](/img/docs/ffa69e1abe35-Picture89.png) **Bước 2.** Thực hiện chọn version muốn upgrade: ![](/img/docs/bebb276a44d6-Picture90.png) **Bước 3.** Nhấn **update** để thực hiện Upgrade version. ![](/img/docs/cd0fd45c6bf7-Picture91.png) **Bước 4.** Kết quả sau khi upgrade version thành công ![](/img/docs/5539c4c1d4ac-Picture92.png) **Lưu ý:** Hệ thống sẽ mất một khoảng thời gian để upgrade version.
