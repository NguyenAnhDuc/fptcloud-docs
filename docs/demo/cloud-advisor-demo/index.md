# Cloud Advisor
Cloud Advisor giúp bạn chủ động phát hiện và xử lý các vấn đề về security, chi phí và độ ổn định của hệ thống FPT Cloud — thay vì chờ đến khi có sự cố.
## Tính năng chính
- 🔒 Security — Phát hiện cấu hình sai, MFA chưa bật, Security Group mở rộng
- 💰 Cost Optimization — Tìm resource lãng phí, instance nhàn rỗi
- 🛡️ Fault Tolerance — Kiểm tra snapshot, backup còn thiếu
## Điều kiện tiên quyết
- Tài khoản FPT Cloud với quyền Admin hoặc Editor
## Bắt đầu sử dụng
1. Đăng nhập vào FPT Cloud Console
1. Chọn Cloud Advisor từ menu điều hướng bên trái
1. Xem Dashboard tổng quan — Action recommended và Investigation recommended
1. Click vào từng category để xem chi tiết vi phạm
1. Với mỗi vi phạm: chọn Fix ngay, Exclude (nếu cố ý), hoặc Download báo cáo
> 💡 💡 Lưu ý: Dữ liệu Cloud Advisor refresh theo chu kỳ, không phải real-time. Nhấn nút Refresh để cập nhật thủ công.
## Nội dung liên quan
- Dashboard — Hướng dẫn đọc các chỉ số
- Security — Chi tiết các rule bảo mật
- Cost Optimization — Chi tiết các rule tiết kiệm chi phí
- Fault Tolerance — Chi tiết các rule về độ ổn định
