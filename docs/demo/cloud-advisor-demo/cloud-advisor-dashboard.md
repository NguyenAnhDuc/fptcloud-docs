# Cloud Advisor Dashboard
Dashboard hiển thị tổng quan các vấn đề cần xử lý theo từng category.
## Các chỉ số trên Dashboard
- Action recommended: Số vấn đề cần xử lý ngay
- Investigation recommended: Vấn đề cần kiểm tra thêm
- Rules with excluded items: Các rule đã loại trừ
> 💡 💡 Mỗi vấn đề có nút Download để xuất báo cáo Excel cho báo cáo nội bộ.
