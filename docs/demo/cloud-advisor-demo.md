# Cloud Advisor
Cloud Advisor giúp bạn phát hiện các vấn đề về security, chi phí và độ ổn định của hệ thống.
## Bắt đầu sử dụng
1. Đăng nhập vào FPT Cloud Console
1. Chọn Cloud Advisor từ menu bên trái
1. Xem Dashboard tổng quan
