# 2. FPT ArgoCD


Change password


* * *

Hệ thống cho phép người dùng change password cho account đã tạo:

_Lưu ý: Người dùng chỉ có thể change password khi quên password, không thể thực hiện copy password từ portal._

**Bước 1** : Tại màn hình**Security & Access** > **Local Account** > **Change Password** :

![](/img/docs/2d97201cdc54-Picture88.png)

**Bước 2** : Thực hiện edit password:

![](/img/docs/37eec2357d5d-Picture89.png)

Bước 3: Click **Change Password** để hoàn thành việc cấu hình.
