# 2. FPT ArgoCD


Update admin password


* * *

FPT Cloud hỗ trợ người dùng thay đổi admin password theo yêu cầu như sau:

  * **Bước 1:** Tại màn **Essential Properties > Click** Update Password:** ![](/img/docs/144af9795134-Picture69-2.png)

  * **Bước 2:** Sau khi Click Update Password, thực hiện nhập thông tin password muốn thay đổi: ![](/img/docs/6f624b542676-Picture70-2.png)

  * **Bước 3:** Kết quả sau khi Update password: ![](/img/docs/2fd47aa89728-Picture71-2.png)

Sau khi update password sử dụng password mới để thực hiện login vào ArgoCD
