# 2. FPT Data Platform


Xoá Nessie


* * *

Để xóa Nessie, người dùng thực hiện các bước sau:

**Bước 1** : Tại thanh menu chọn **Data Platform** > chọn **Workspace Management** > chọn **Workspace name**

**Bước 2** : Tại phần application chọn **Nessie** > nhấn vào nút action góc phải màn hình chọn **Delete**

![anh-3](/img/docs/999dafde0b19-imaged.png)

**Bước 3** : Hiển thị hộp thoại **Delete** application > nhập delete > nhấn confirm để xóa hoàn thành việc xóa app

![anh-3](/img/docs/04ffdeebdf00-imagee.png)
