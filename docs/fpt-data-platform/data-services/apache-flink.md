# 2. FPT Data Platform


Flink


* * *

Apache Flink là một **framework xử lý dữ liệu phân tán** mã nguồn mở, chuyên dụng cho **xử lý luồng dữ liệu (stream processing)** theo thời gian thực. Ngoài ra, nó cũng hỗ trợ **xử lý theo lô (batch processing)** nhưng nổi bật nhất ở khả năng **xử lý dữ liệu liên tục với độ trễ thấp**. 

Flink có khả năng mở rộng linh hoạt, hỗ trợ trạng thái (stateful processing) và đảm bảo **tính nhất quán** , giúp nó trở thành lựa chọn hàng đầu trong các hệ thống **phân tích dữ liệu lớn (Big Data Analytics), Machine Learning, IoT, tài chính, và giám sát hệ thống**.
