# 2. FPT Data Platform


Xóa Open Metadata


* * *

Để xóa **Open Metadata** **service** , người dùng thực hiện các bước sau:

**Bước 1:** Tại thanh menu chọn **Data Platform** > chọn **Workspace Management** > chọn **Workspace name**

**Bước 2:** Chọn **Open Metadata** > nhấn vào nút **Action** góc phải màn hình chọn **delete**

![Hình ảnh](/img/docs/b77d215ec7d1-image32.png)

**Bước 3** : Hiển thị hộp thoại Delete application > nhập delete > nhấn Confirm để xóa hoàn thành việc xóa app 

![Hình ảnh](/img/docs/f281dc909013-image3f.png)
