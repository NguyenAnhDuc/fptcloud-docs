# 2. FPT Data Platform


Phân quyền người dùng sử dụng profile


* * *

Sau khi đăng nhập Jupyterhub, người dùng có vai trò **Admin** lựa chọn Menu **Service** > **Profile** , ấn tab **Profile Assignments**

Hệ thống hiển thị thẻ tương ứng với các profile đã tạo, ấn vào profile cần phân quyền để mở ra thông tin 

  * **Add User** : Lựa chọn từ danh sách User trong hệ thống 

  * **Add Group** : Lựa chọn từ danh sách Group trong hệ thống 

Sau khi lựa chọn, ấn **Save Changes** để tạo phân quyền người dùng/nhóm vào profile
