# My Account


* * *

# 1\. How to manage your API Keys?

## 1.1 How to view your API Keys?

To check your API keys, please follow the instruction below:

**Note** : Please ensure your account is subscribed to a paid plan. If you don’t have one, please [**contact us**](<https://fptcloud.com/en/documents/ai_marketplace/?doc=contact-us>).

Step 1: Click on the account icon on the header. ![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/0.png) Step 2: Select **My Account.**

Your list of API keys will be shown on the **My API Keys** page. ![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/my-api-key.png)

## 1.2 How to create a new API Key?

To create an API Key, please follow these steps:

Step 1: Go to the **My API Keys** page and click **Create New API Key** button.

Step 2: Input **Name**

Step 3: Choose the models you’d like to access.

Step 4: Click **Create** button to generate your new API Key. ![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/create-my-api-key.png) After your new API Key created, please copy or save your key.

_Note: The API key is only visible and can be copied once at creation. Please save your key in safe place and do not share your API key with others, or expose it in the browser or other client-side code, anyone with your API Key can make requests on your behalf.In order to protect the security of your account, AI Marketplace may automatically disable any API key that we've found has leaked publicly._

Then, click the **Done** button to finish. ![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/done.png)

## 1.3 How to edit an existing API Key?

You can edit **Name** and **Permission** of the API Key. To edit an API Key, please follow these steps: 

**Step 1** : Go to the **My API Keys** page and click Action next to the API Key you want to edit. 

**Step 2** : Select **Edit** from the dropdown menu. 

**Step 3** : Edit your information. 

**Step 4** : Click Save button to finalize the changes. 

## 1.4 How to revoke an API Key?

Before revoking the API Key, please note: After revoking, this API key will immediately be disabled. API requests made using this key will be rejected, which could cause any systems still depending on it to break. 

To revoke an API key, please follow the following steps: 

**Step 1** : Go to the **My API Keys** page and click **Action** next to the API key you want to revoke. 

**Step 2** : Select **Revoke** from the dropdown menu. 

**Step 3** : Click **Revoke key** button to confirm and delete the API key. ![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/revoke.png)

# 2\. How to check your usage?

To check your usage including Input tokens, Output tokens, Total Usage for each Model and API Key, please go to **My Account > My usage** ![Alt text](https://fptcloud.com/wp-content/uploads/2025/02/my-usage.png)
