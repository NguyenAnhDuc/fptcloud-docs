# 2. FPT Container Registry


Disable Robot Account đang sử dụng 


* * *

**Bước 1** : Tương tự như thực hiện Edit Robot Account, tích chọn **Robot Account** muốn disable, sau đó chọn **ACTION DISABLE** : 

![](/img/docs/5d9785b075bd-Picture69.png)

**Bước 2** : Trên Portal xuất hiện hộp thoại để xác nhận hành động. Nhập **DISABLE** để thực hiện disable account: ![](/img/docs/e994d83323e6-Picture70.png)

**Bước 3** : Kết quả sau khi thực hiện disable, **Enabled State** sẽ hiển thị như hình bên dưới. 

![](/img/docs/071816c735e2-Picture71.png)
