# 2. FPT Container Registry


Xóa Image hoặc Artifact trong Repositories


* * *

Trong trường hợp bạn muốn xóa Image hoặc Artifact đã push lên trước đó để tiết kiệm dung lượng hoặc loại bỏ các project rác, hãy thao tác theo các bước sau: 

**_1\. Xoá Repository_**

**Bước 1** : Ở phần **Action** của Image hoặc Artifact chọn **Delete.** ![](/img/docs/efab785ae718-Picture19-1.png)

**Bước 2** : Chọn Delete trong popup Confirmation. ![](/img/docs/a67dbf3a8a0a-Picture20-1.png)

**_2\. Xoá Artifact_**

**Bước 1** : Lấy thông tin Detail Repository

![](/img/docs/14112c716f1b-Picture21-1.png)

**Bước 2** : Ở phần Action của Artifact, chọn Delete 

![](/img/docs/812a26869735-Picture22-1.png)

**Bước 3** : Chọn Delete trong popup confirmation

![](/img/docs/7da45a001279-Picture23-1.png)

**_3\. Xoá Tag_**

**Bước 1** : Lấy thông tin Detail Tag 

![](/img/docs/65f144c5eeb3-Picture24-1.png)

**Bước 2** : Ở phần Action của Tag, chọn Delete 

![](/img/docs/609bcb945ac9-Picture25.png)

**Bước 3** : Chọn Delete trong popup confirmation
