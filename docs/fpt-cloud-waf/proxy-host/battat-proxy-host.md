# 2. FPT Cloud WAF


Bật/Tắt Proxy Host


* * *

**Bước 1:** Ở **WAF Dashboard** , chọn **Hosts** > **Proxy**.

![Userguide FPT WAF 2022 17](/img/docs/72760d67e188-Userguide-FPT-WAF-2022-17-1024x578.png)

**Bước 2:** Ở **Proxy** cần thay đổi trạng thái Bật/Tắt chọn icon **More** > Chọn chức năng **Toggle**.

![Userguide FPT WAF 2022 22](/img/docs/a1462ad18c9c-Userguide-FPT-WAF-2022-22-1024x578.png)

**Bước 3:** Hộp thoại xác nhận hiện lên. Chọn **Run Action** để thực hiện thao tác**.**

![Userguide FPT WAF 2022 23](/img/docs/48fa2290aefe-Userguide-FPT-WAF-2022-23.png)
