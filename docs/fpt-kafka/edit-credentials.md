# Edit Credentials


* * *

The system allows you to change the password of a credential. To edit, follow these steps:

**Step 1** : Select the credential that you want to change password > Click **Edit**. 

![](https://fptcloud.com/wp-content/uploads/2023/08/Pic1_sua_cre.png)

**Step 2** : Enter the new password. 

![](https://fptcloud.com/wp-content/uploads/2023/08/Pic2_Sua_cre.png)

**Step 3** : Click **OK** to save the information.
