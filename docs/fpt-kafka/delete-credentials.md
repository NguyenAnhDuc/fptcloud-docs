# Delete Credentials


* * *

In order to delete a credential, follow the steps:

**Step 1** : Select the credential to delete > Click **Remove**.

![](https://fptcloud.com/wp-content/uploads/2023/08/Pic1_XoaCre.png)

**Step 2** : In the confirmation dialog, select **Yes** to confirm deletion.

![](https://fptcloud.com/wp-content/uploads/2023/08/Pic2_remove.png)
