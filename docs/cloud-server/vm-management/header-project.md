# Thay đổi giao diện header


* * *

Phân quyền: Người dùng hiện có quyền truy cập vào VPC nào thì sẽ thấy danh sách chứa VPC đó.

![file](/img/docs/83d850826a59-image-1733738037369.png)

### **1\. Ẩn thông tin username, người dùng có thể nhấn vào icon dropdown để xem thông tin cá nhân**

![file](/img/docs/12ac0defadc0-image-1733738050652.png)

### **2\. Chuyển filter theo Tenant từ góc trái sang góc phải bên cạnh hình đại diện của người dùng.**

![file](/img/docs/37904692f8c7-image-1733738066401.png)

![file](/img/docs/757c1855bc0a-image-1733738071877.png)

### **3\. Hiển thị menu support dưới dạng danh sách các icon, người dùng hover chuột vào icon sẽ hiện thông tin**

![file](/img/docs/3164cea943fa-image-1733738084499.png)

### **4\. Thêm filter Project**

  * Default hệ thống hiển thị All project

  * User có thể chọn project để filter VPC bên dưới 

Lưu ý: Hệ thống không hiển thị project không có VPC lên danh sách trên header
