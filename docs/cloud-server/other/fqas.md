# FQAs


* * *

**1\. Tại sao một số token tôi không thể revoke được?**

Đáp: Một số token cũ được khởi tạo trước bản nâng cấp của FPT Cloud Portal sẽ không thể revoke. Đối với token đó, hệ thống sẽ thông báo bằng tooltip với nội dung: “Tokens created earlier cannot be revoked”

**2\. Token chuyển trạng thái “Expired” khi nào?**

Đáp: Khi token vượt qua thời gian “Expiration” tính từ lúc khởi tạo. Hệ thống sẽ tự động chuyển token sang trạng thái “Expired” và người dùng sẽ không thể truy cập bằng token
