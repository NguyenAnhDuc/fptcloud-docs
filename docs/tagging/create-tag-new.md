# 2. Tagging


Khởi tạo một tag mới 


* * *

Người dùng thao tác như sau:

**Bước 1** : Ở menu chọn **Tagging** , chọn **Create tag**

![file](/img/docs/a69582402ad0-image-1721012726449.png)

  * Bước 2: Nhập các thông tin sau: Tag Level: Tenant, Project, VPC

  * Resource Scope: đối với level Project/VPC, hệ thống có Tag Level: Tenant, Project, VPC 

  * Resource Scope: đối với level Project/VPC, hệ thống cho phép chọn resource scope tương ứng. 

![file](/img/docs/b41d710f8726-image-1764325536204.png)

Bước 3: Nhấn **Create** để tạo tag. Tag sau khi được khởi tạo sẽ được hiển thị ở trang Tagging

![file](/img/docs/5a0fdb03c0a6-image-1764325568380.png)
