# 2. Trellix Customer Guide


Quản lý các tác vụ chạy trên máy chủ (Client Tasks)


* * *

**Bước 1** : Chọn thanh menu > System Tree > Group > Assigned Client Task

![file](/img/docs/189ffc4dfd31-image-1765765586369.png)

**Bước 2** : Người dùng có thể xem các tác vụ đang được lên lịch và chạy trên các máy chủ đã cài thành công agent.

![file](/img/docs/da57bd94129f-image-1765765604165.png)

  * Update AMCore Content Package: Cập nhật các bản signature mới nhất – daily

  * Upgrade Agent – Linux: Kiểm tra và cập nhật lên các phiên bản agent mới nhất trên Linux OS – weekly

  * Upgrade Agent – Windows: Kiểm tra và cập nhật lên các phiên bản agent mới nhất trên Windows OS/Windows Server – weekly

  * Customer- Full Scan: Tác vụ rà quét mã độc trên máy chủ - weekly

  * Upgrade Endpoint Security – Linux: Kiểm tra và cập nhật lên các phiên bản ENS mới nhất trên trên Linux OS - monthly

  * Upgrade Endpoint Security – Linux: Kiểm tra và cập nhật lên các phiên bản ENS mới nhất trên trên Windows OS/Windows Server - monthly

  * Linux/Windows EndPoint Security Deploy: Cài đặt các Enpoint Security ngay lập tức khi sau khi có kết nối giữa agent và server (sau khi cài đặt thành công agent)
