# 2. FPT App Catalogs


Thực hiện uninstall applications nếu muốn xoá bỏ


* * *

Hệ thống hỗ trợ người dùng xoá bỏ chart đã được cài đặt trên cluster

**Bước 1** : Tại màn hình detail cluster thực hiện click button “**Uninstall** ”

![](/img/docs/825b929dead7-Picture35-1.png)

**Bước 2** : Nhập “**uninstall** ” để xác nhận việc gỡ chart

![](/img/docs/9e7c9fb32931-Picture36-1.png)

**Bước 3** : Click “**Uninstall** ” để thực hiện gỡ chart và kiểm tra kết quả hiển thị

![](/img/docs/566a9f4cecd4-Picture37-1.png)
