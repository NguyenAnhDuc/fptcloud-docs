# 2. FPT App Catalogs


Search app catalogs trong repository


* * *

Để tìm kiếm app catalogs có thể thực hiện như sau:

  * **Bước 1** : Tại màn hình **App Catalogs** , thực hiện tìm kiếm chart muốn deploy

  * **Bước 2** : Nhập thông tin muốn tìm kiếm

Ví dụ chọn repository là bitnami, thực hiện tìm kiếm chart cert-manager của bitnami để deploy tới cluster.
