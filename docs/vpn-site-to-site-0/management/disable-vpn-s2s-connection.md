# Disable VPN-Site-to-Site


* * *

Vô hiệu hóa VPN đang hoạt động (nếu VPN đang Active).

**Bước 1** : Chọn kết nối VPN cần Disable.

![Config 15](/img/docs/574882a19472-Config-15-1024x262.png)

![Disable VPN](/img/docs/0338f03770d6-Disable-VPN-1024x354.png)

**Bước 2:** Xác nhận việc Disable. 

![Disable VPN confirm](/img/docs/a8ee08e49340-Disable-VPN-confirm-1024x445.png)

**Bước 3:** VPN sẽ chuyển sang trạng thái **Deactivating** > **Inactive**. 

![Config inactive](/img/docs/c1dec1de2cba-Config-inactive-1024x238.png)
