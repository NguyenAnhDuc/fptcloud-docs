# Enable VPN-Site-to-Site


* * *

Enable: Cho phép VPN hoạt động (nếu VPN đang Inactive).

**Bước 1** : Chọn kết nối VPN cần Enable. 

![Config choose more action 1](/img/docs/029a47acf3bf-Config-choose-more-action-1-1024x238.png)

![Config enable](/img/docs/dcdc17283275-Config-enable-1024x354.png)

**Bước 2:** Xác nhận việc Enable. 

![Config confỉrm enable](https://fptcloud.com/wp-content/uploads/2023/05/Config-confỉrm-enable-1024x445.png)

**Bước 3** : VPN sẽ chuyển sang trạng thái **Activating** > **Active**. 

![Config active](/img/docs/e65f8da446c3-Config-active-1024x238.png)
