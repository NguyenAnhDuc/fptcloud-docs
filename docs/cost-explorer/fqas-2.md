# 2. Cost Explorer


Tôi muốn xem Cost Explorer của 1 vài Instance hoặc Security Group nhưng Customize Filter không cho phép chọn ?


* * *

Customize Filter của Cost Explorer cho phép bạn filter theo Tag. Sử dụng tính năng Tagging để đánh tag các tài nguyên cụ thể mà bạn muốn xem chi phí, sau đó filter Cost Explorer theo tag này.
