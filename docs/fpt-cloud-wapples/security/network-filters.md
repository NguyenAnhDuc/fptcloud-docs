# 2. FPT Cloud WAPPLES


Network Filters


* * *

Sử dụng để cấu hình network (tương tự security group)

  * Các trường hiển thị
  * Filter No.: Hiển thị số hiệu của policy
  * Source IP: ip truy cập
  * Source port: port truy cập
  * Source netmask: subnet được truy cập
  * Destination IP: ip đích dịch vụ được truy cập đến
  * Destination Port: port đích dịch vụ
  * Destination Netmask: subnet đích được truy cập 
  * Protocol: Giao thức truy cập
  * Connection: Hành động thực hiện khi phát hiện kết nối(Allow, Deny, Detection)
