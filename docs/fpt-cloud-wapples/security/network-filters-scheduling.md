# 2. FPT Cloud WAPPLES


Network Filters Scheduling


* * *

Sử dụng thiết lập thời gian thực hiện các security group trong network filter

Các trường cần điền

  * **Schedule name**
  * **Schedule description**
  * **Day of Week** : Lựa chọn thời gian thực hiện. Có thể chọn từng ngày hoặc chọn nhanh các ngày trong tuần hoặc cuối tuần
  * **Start Time và End Time** : Thời gian bắt đầu và kết thúc muốn Security Group hoạt động trong ngày ![file](/img/docs/de7cc26dc083-image-1714115260868.png) ![file](/img/docs/844c70bae31e-image-1714115269843.png)
