# 2. FPT Cloud WAPPLES


Destination Server


* * *

Chức năng thực hiện thiết lập web service từ các ip host nằm trong cloud ![file](/img/docs/4364137ed761-image-1714121626519.png)

  * Thêm/sửa/xóa Destination server ![file](/img/docs/5890dbc9d717-image-1714719775318.png)

  * Mode: Proxy

  * Proxy IP: IP Proxy đã được thiết lập

  * Proxy Port: port proxy được chuyển đổi để sử dụng từ port gốc

  * Web Server IP (Domain): ip hoặc domain service của webserver

  * Web Server Port: port service của webserver

  * Health Check: kiểm tra trạng thái của webserver

![file](/img/docs/ec1e303d65a4-image-1714719898104.png)

  * Gateway: IP Gateway được thiết lập

  * VLAN ID: Hiển thị VLAN ID được nhập. Trống nếu không dùng SSL

  * SSL Use: Hiển thị X nếu không dùng, O nếu sử dụng

  * SSL Termination: Hiển thị X nếu không dùng, O nếu dùng

  * Certificate Expiration: Hiển thị hạn của chứng chỉ được chọn nếu sử dụng SSL
