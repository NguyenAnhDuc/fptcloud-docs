# 2. FPT Cloud WAPPLES


Proxy IP


* * *

Chức năng thực hiện thiết lập địa chỉ proxy IP nếu có sử dụng nhằm thiết lập các dịch vụ của ip private trong cloud ![file](/img/docs/859e05760757-image-1714112480774.png)

  * Các chức năng có thể sử dụng bao gồm thêm, sửa, xoá Proxy IP

  1. Thêm IP Proxy

     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn vào biểu tượng + để thêm Proxy IP
     * Nhập vào các giá trị cần thiết để thiết lập IP như hình và chọn Apply ![file](/img/docs/ca5d7a26f601-image-1714112494432.png)
     * Chọn **Save** để lưu thay đổi ![file](/img/docs/e3275d5b20b6-image-1714112501548.png)
  2. Sửa IP Proxy

     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn vào Modify
     * Thực hiện sửa đổi các thông tin cần thiết: ![file](/img/docs/760a45f00115-image-1714112513079.png)
     * Nhấn chọn Apply và chọn Save để lưu thiết lập ![file](/img/docs/1154d0816895-image-1714115110737.png)
  3. Xoá IP Proxy

     * Tại trang Security Settings/Networks Settings/Proxy IP, chọn vào ip cần xoá ![file](/img/docs/4dfc9591ad66-image-1714115123188.png)
     * Chọn vào biểu tượng thùng rác để thực hiện xoá ip đã chọn và chọn **Save** nhằm thực hiện lưu lại thiết lập. ![file](/img/docs/726201bdf3fb-image-1714115133281.png)
